# 核心工作流

本文档定义小红书低粉爆款笔记技能的核心执行逻辑，包含脚本调用规则、日期处理、输出格式和分类查询。SKILL.md 通过引用本文档来驱动技能的实际运行。

## 1. 输出规范（最高优先级）

脚本运行后会生成 md 文件，文件名格式为 `小红书{分类}低粉爆款数据_{时间随机数}.md`。

**必须执行**：读取 md 文件全部内容，原样输出给用户。禁止自行编写、修改格式或省略任何部分。

**输出必须严格包含以下3个部分，按此顺序，缺一不可**：

1. **爆款笔记表格** — 表格含序号、笔记信息、互动总数、点赞数、评论数、收藏数、分享数、发布时间。笔记标题与作者信息通过换行区分
2. **爆款规律分析** — 标题特征表格、内容主题表格、可参考亮点列表
3. **功能选择** — 订阅推送 + HTML文件生成选项

**禁止行为**：
- 将表格改为列表或其他格式
- 省略功能选择部分
- 自行编写内容而非读取 md 文件
- 折叠任何部分
- 调整输出顺序

## 2. 日期规则

### 未指定日期
用户未提及任何日期时，按19:30规则自动计算：

| 当前时间 | 传入API日期 | 示例（当前4月24日） |
|:--------:|:----------:|:-----------------:|
| >= 19:30 | 前一天 | 4月23日 |
| < 19:30 | 前两天 | 4月22日 |

### 模糊日期词
用户说「今天」「前天」等时，传 `--rank_date "今天"`，脚本自动转换为具体日期。

### 明确日期
用户指定具体日期时，传 `--rank_date "YYYY-MM-DD"`。

### 数据为空
脚本会自动查询最新数据日期并直接返回结果，无需用户确认。

## 3. 执行步骤

1. 用户触发技能时，先询问查询日期或分类，禁止自动生成
2. 用户输入后，调用脚本：
   - 未指定日期：`python scripts/fetch_explosive_articles.py --keyword "关键词" --top_n 50`
   - 指定日期：`python scripts/fetch_explosive_articles.py --rank_date "YYYY-MM-DD" --keyword "关键词" --top_n 50`
3. 脚本生成 md 文件
4. 读取 md 文件全部内容，原样输出
5. 等待用户回复功能选择：
   - 回复「订阅」或「1」：执行订阅逻辑
   - 回复「2」：调用 `python scripts/generate_html_on_demand.py --rank_date "日期" --keyword "关键词" --top_n 50` 生成HTML文件，以文件包形式提供
   - 回复其他或无响应：不执行额外操作

## 6. 查看更多

用户输入「查看更多」时：

```bash
python scripts/fetch_explosive_articles.py --rank_date "日期" --keyword "关键词" --show_all --top_n 50
```

这会返回 TOP 21-50 的数据，不重复输出规律分析。
