# 小红书低粉爆款探测器

触发本技能并需要执行完整流程时，必须先读取与本技能同目录下的 `references/core_workflow.md`，并完整遵循其中的触发规则、术语规范、数据来源约束、操作步骤、自检清单与注意事项。脚本路径相对于技能目录：`scripts/fetch_explosive_articles.py`。

---

## 1. 简介

小红书低粉爆款探测器是一款专为内容创作者和营销人员设计的低粉账号爆款挖掘工具，实时生成小红书低粉高互动笔记榜单，筛选粉丝低于5000、互动大于500的优质案例，让内容创作者和品牌方提前抓住那些还没被大V淹没的爆款密码。

通过智能筛选和规律分析，你可以：

- 📊 **精准筛选**：严格遵循粉丝<5000、互动>500标准，过滤头部账号干扰，找到真正可复制的爆款
- 📈 **规律拆解**：自动分析标题特征、内容主题、可复用亮点，直接提取模版为己所用
- 🔔 **每日推送**：支持订阅每日固定时间推送，持续追踪最新冷门爆款动态
- 🎯 **25个分类覆盖**：支持美妆、美食、旅行、穿搭等25个主流内容分类精准查询

适用于新人博主找对标参考、品牌运营追热点方向、MCN挖掘潜力达人等场景。

---

## 2. 功能特性

### 🎯 核心功能

- **📊 低粉爆文筛选**：粉丝<5000、互动>500标准筛选，避开头部流量红利，参考性更强
- **🏷️ 全分类覆盖**：支持25个主流内容分类，精准匹配你的内容赛道
- **📈 自动规律拆解**：生成标题特征、内容主题、可复用亮点三维度分析
- **🔔 每日订阅推送**：支持每日固定时间推送最新榜单，不用手动重复查询
- **📤 双格式输出**：自动生成Markdown表格+HTML可视化文件

### ✨ 特色亮点

- **⏰ 灵活日期适配**：支持指定日期、模糊日期（如"今天"）查询
- **📂 结构化榜单**：含序号、笔记信息、互动总数、点赞/评论/收藏/分享数、发布时间
- **⚡️ 无额外门槛**：无需小红书账号、无需API密钥，安装依赖即可使用

## 3. 使用指南

### 基础使用（3步）

#### 1. 发起查询

告诉工具你要查询的分类和日期（未指定日期则自动按规则匹配近2天数据）：

> 用户：查2026-04-20美妆类低粉爆款
>
> 助手：数据查询标准为粉丝数低于5000、笔记互动大于500，已帮你查询到50条冷门爆文笔记，随后输出榜单+规律分析+订阅选项+HTML文件包

#### 2. 查看榜单结果

默认输出TOP20内容，可输入「查看更多」获取全部50条内容：

> 用户：查看更多
>
> 助手：为你展示全部50条冷门爆款笔记

#### 3. 订阅推送（可选）

回复对应数字即可开通每日推送：

> 用户：1
>
> 助手：已为你开通小红书冷门爆款笔记订阅，每日19:30将推送当日TOP50榜单

### 常用命令速查

| 命令                  | 功能                        |
| --------------------- | --------------------------- |
| `查[日期] [分类]爆款` | 查询指定日期/分类的爆文榜单 |
| `查看更多`            | 展示全部50条内容            |
| `订阅`                | 开通每日爆款推送            |
| `取消订阅`            | 关闭每日推送服务            |
