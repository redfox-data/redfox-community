# AI 图片生成器 / gpt-image2-pro

---

## 简介

基于 gpt-image-2 模型，输入提示词即可生成高质量图片，支持文生图与图生图，内置免费额度开箱即用。

**核心价值**

- **开箱即用**：内置公共额度约 10000 次，无需额外订阅或绑定信用卡，粘贴提示词就能用
- **双模式覆盖**：文生图从零创作，图生图基于参考图编辑，一个参数 `--image` 即可切换
- **灵活可控**：支持多种尺寸、格式、透明背景与保真度参数，批量生成一次最多 10 张

**适用对象**

- 🎨 **设计师 / 创作者** — 快速出图、风格探索，降低视觉创作门槛
- 📢 **品牌 / 电商运营** — 批量生成产品图、透明背景 logo，统一视觉风格
- 💡 **内容策划** — 为推文、笔记、短视频配图，一键获取可用素材

---

## 功能特性

### 核心功能

- **文生图**：输入文字提示词，生成全新图片
- **图生图**：上传参考图 + 修改提示词，基于原图编辑生成新图
- **多种尺寸**：支持 1024×1024、1024×1792、1792×1024 及自动尺寸
- **批量生成**：单次最多生成 10 张，方便对比与筛选
- **透明背景**：支持 PNG / WebP 格式透明背景输出
- **保真控制**：图生图模式支持高保真（微调）与低保真（大幅改动）两种模式
- **多格式输出**：PNG、JPEG、WebP 三种格式可选，支持压缩比调节

### 特色亮点

- **内置免费额度**：约 10000 次调用，无需配置即可开始使用
- **零配置起步**：脚本内置公共 API Key，首次使用无需任何设置
- **额度续用**：免费额度用完后，前往红狐 hub 注册即可获取自己的 Key 继续使用

- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述你想要的图片即可，无需记忆命令。

### 常用说法速查

| 意图       | 示例话术                                         | 效果                         |
| ---------- | ------------------------------------------------ | ---------------------------- |
| 文生图     | 「帮我画一只橘色的猫咪坐在窗台上看着窗外的夕阳」 | 根据提示词生成全新图片       |
| 指定横版   | 「生成一张未来城市天际线，横版高清」             | 按横版尺寸生成               |
| 透明背景   | 「帮我生成一个极简风格的猫 logo，透明背景」      | 输出透明背景 PNG 图片        |
| 批量生成   | 「帮我生成 4 个扁平风格图标」                    | 一次生成 4 张，方便对比      |
| 图生图编辑 | 「把这张图里的猫咪改成白色，背景换成星空」       | 上传参考图，按提示词编辑生成 |
| 风格迁移   | 「把这张照片改成赛博朋克风格」                   | 基于原图做风格变换           |
| 大幅改动   | 「把这张照片变成水彩画风格，加入樱花元素」       | 低保真模式，允许大幅修改     |

### 输出示例

生成完成后，图片将自动保存到本地目录，你将收到类似以下结果：

**文生图示例**

提示词：「一只橘色的猫咪坐在窗台上看着窗外的夕阳」

→ 生成 1 张 1024×1024 PNG 图片，保存至 `~/Downloads/QoderImages/`

**图生图示例**

提示词：「把猫咪改成白色，背景换成星空」+ 参考图

→ 基于原图编辑生成新图，保留构图的同时按提示词修改
