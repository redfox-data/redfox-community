# 抖音热榜神器 / douyin-hot-trend

---

## 简介

实时追踪抖音热点、挖掘爆款规律，帮助内容创作者和运营者高效蹭热点、找选题的智能热榜工具。

**核心价值**

- 每小时自动更新抖音 TOP 50 榜单，热度值与跳转链接一键可见
- 覆盖近 7 天、近 30 天历史热榜，随时回溯过往热点演变
- 从创作者视角自动生成选题机会、标题套路与趋势预判
- 支持任意两个时间点热榜对比，清晰标注新进榜与掉榜话题
- 支持每小时或每日定时推送，无需手动触发

**适用对象**

- 🎬 **短视频创作者** — 每天快速锁定热点方向，缩短选题时间
- 📣 **MCN 机构运营** — 统一监控平台趋势，高效指导达人内容策略
- 🏷️ **品牌营销人员** — 发现可借势话题，提升热点响应速度
- 📚 **运营新手** — 学习爆款标题套路，快速建立内容感知

---

## 功能特性

### 核心功能

- **实时热榜查询**：获取抖音当前 TOP 50 热榜，包含排名、热度值、话题标题和跳转链接，每 60 分钟刷新
- **历史热榜回溯**：支持查询近 7 天或近 30 天历史数据，可按日期指定查看特定某天
- **热榜对比**：对比任意两个时间点的热榜排名与热度变化，自动标注新进榜与掉榜话题
- **创作者洞察报告**：热榜展示后自动输出爆款选题机会、标题套路拆解、趋势预判与行动建议
- **可视化报告导出**：一键生成紫色极简风格 HTML 页面，支持导出 PDF，每条话题可点击跳转
- **定时订阅推送**：支持每小时或每日定时推送，自动追踪热点变化

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图         | 示例话术               | 效果                                                  |
| ------------ | ---------------------- | ----------------------------------------------------- |
| 查看实时热榜 | 抖音热榜 / 今日热榜    | 展示当前 TOP 20，含热度值与话题链接，附创作者洞察报告 |
| 查看昨日热榜 | 昨日热榜               | 查询昨天全天热榜数据                                  |
| 查看历史热榜 | 近7天热榜 / 近30天热榜 | 按时间范围回溯历史热点                                |
| 查看指定日期 | 5月15日热榜            | 查询特定日期的榜单数据                                |
| 加载完整榜单 | 继续加载 / 加载更多    | 从 TOP 20 扩展至完整 TOP 50                           |
| 对比热榜变化 | 对比昨天和今天的热榜   | 输出两日热榜差异，标注新进榜与掉榜话题                |
| 订阅推送     | 订阅每日热榜推送       | 设置定时自动推送，每日获取最新热榜                    |
| 取消订阅     | 取消订阅               | 停止定时推送                                          |

### 输出示例

查询「今日热榜」后，你将收到：

1. **热榜表格**：序号、话题标题、热度值、跳转链接（TOP 20，可继续加载至 50）
2. **创作者洞察报告**：
   - 🔥 爆款选题机会（可迁移话题与创作方向）
   - 🎯 标题套路拆解（情绪词、句式结构）
   - 📈 趋势预判（哪些话题正在上升/下降）
   - 💡 行动建议（可执行的内容策略）
3. **可视化报告**：自动生成 HTML 页面，支持导出 PDF
