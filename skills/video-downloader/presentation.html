07

常见问题

公共 Key 有多少次免费额度？
约 10000 次。超出后 API 返回 3107 错误，需配置自己的 Key。
如何获取自己的 API Key？
前往 redfox.hk 注册获取，格式为 ark_xxx。
下载的视频有水印吗？
没有。API 返回的是无水印直链，画质与原始作品一致。
支持图文下载吗？
支持图文下载（暂不支持小红书图文）。自动下载所有图片，按序号命名。
