# 短视频下载器 / video-downloader

---

## 简介

粘贴分享链接，一键下载抖音、小红书、快手、B站无水印视频/图文，内置公共 API Key，开箱即用。

**核心价值**

- **无水印直链**：API 自动返回无水印下载地址，无需手动去水印。
- **四平台覆盖**：一个工具搞定抖音、小红书、快手、B站。
- **零配置启动**：内置公共 API Key（约 10000 次免费额度），直接用。

**适用对象**

- 🎬 **短视频创作者** — 下载素材用于二次创作和剪辑。
- 📱 **内容运营** — 批量收藏竞品视频，分析热门内容。
- 💾 **普通用户** — 保存喜欢的视频到本地，离线观看。

---

## 功能特性

### 核心功能

- **无水印下载**：API 自动返回无水印直链，无需手动处理。
- **图文下载**：支持图文类型下载所有图片（暂不支持小红书图文）。
- **跨平台支持**：一个工具覆盖抖音、小红书、快手、B站四个平台。
- **链接自适应**：支持短链、分享链、网页链接等多种格式自动识别。
- **进度显示**：下载过程实时显示进度条和百分比。
- **多端兼容**：支持手机端分享链接和 PC 端网页链接。

## API Key 说明

内置公共 API Key，约 **10000 次免费额度**，零配置直接使用。额度用完后可前往 [RedFoxHub](https://redfox.hk/login?source=github) 注册获取个人 API Token：

| 方式 | 命令 |
|------|------|
| **环境变量**（推荐） | `export REDFOX_API_KEY=ak_你的密钥` |
| **命令行参数** | `python3 "$SKILL_PATH/assets/downloader.py" "<链接>" --api-key ak_你的密钥` |
| **配置文件** | `echo '{"api_key":"ak_你的密钥"}' > ~/.qoder/apis/redfox.json` |

## 使用指南

粘贴分享链接即可下载，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 下载抖音视频 | 「帮我下载这个抖音视频 https://v.douyin.com/xxx」 | 自动解析并下载无水印视频 |
| 下载小红书视频 | 「下载这个小红书 http://xhslink.com/o/xxx」 | 下载无水印视频到本地 |
| 下载B站视频 | 「帮我保存这个B站视频 https://b23.tv/xxx」 | 下载最佳画质视频 |

### 支持平台与链接格式

| 平台 | 链接格式 | 说明 |
|------|----------|------|
| 抖音 | `https://v.douyin.com/xxxxxx/` | 手机分享链接 / PC 网页链接 |
| 小红书 | `http://xhslink.com/o/xxxxxx` | 短链分享链接 |
| 快手 | `https://v.kuaishou.com/xxxxxx` | 手机分享链接 |
| B站 | `https://b23.tv/xxxxxx` | 短链分享链接 |
