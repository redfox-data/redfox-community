# 短视频下载器

通过 [redfox.hk](https://www.redfox.hk) API 解析并下载抖音、小红书、快手、B站的无水印视频。支持手机端和 PC 端的分享链接，开箱即用。

---

## 能力概述

支持从主流短视频平台解析作品链接，获取无水印的原始视频。

- **平台支持**：抖音、小红书、快手、B站
- **内容类型**：视频（mp4）、图文（jpg/png，暂不支持小红书图文）
- **输入方式**：粘贴分享链接即可（每次仅限一个链接，不支持批量上传）
- **去水印**：API 自动返回无水印直链
- **开箱即用**：内置公共 API Key，无需任何配置即可使用

## 配置安装

### API Key 配置

| 配置方式 | 说明 | 命令 |
|----------|------|------|
| **内置公共 Key**（开箱即用） | 免费约 10000 次，超出后需自行配置 | `python3 "$SKILL_PATH/assets/downloader.py" "<分享链接>"` |
| **环境变量**（推荐） | 设置一次，全局生效 | `export X_API_KEY=ark_你的密钥` |
| **命令行参数** | 临时使用，单次生效 | `python3 "$SKILL_PATH/assets/downloader.py" "<链接>" --api-key ark_你的密钥` |
| **配置文件** | 持久化存储，跨会话保留 | `mkdir -p ~/.qoder/apis && echo '{"api_key":"ark_你的密钥"}' > ~/.qoder/apis/redfox.json` |

> API Key 绑定用户身份（vid），需要用户身份的接口会自动注入，无需手动传 vid。Key 获取地址：[www.redfox.hk](https://www.redfox.hk)

## 常见问题

**Q：公共 Key 有多少次免费额度？**
A：约 10000 次。超出后 API 会返回 3107 错误，请配置自己的 Key。

**Q：如何获取自己的 API Key？**
A：前往 [www.redfox.hk](https://www.redfox.hk) 注册即可获取，格式为 `ark_xxx`。

**Q：下载的视频有水印吗？**
A：没有。API 返回的是无水印直链。

**Q：支持图文下载吗？**
A：支持图文类型下载（暂不支持小红书图文）。图文内容会自动下载所有图片，并按序号命名。

**Q：可以批量上传多个链接吗？**
A：不支持。每次只能输入一个链接，批量上传会导致解析失败。如需下载多个视频，请逐个粘贴链接。

**Q：链接提示解析失败怎么办？**
A：确认链接是否完整、是否已过期。短链有时效性，建议重新复制分享链接。

**Q：下载速度慢怎么办？**
A：大文件下载需要稳定的网络连接，可尝试更换网络环境。
