# 小红书评论分析 / xiaohongshu-comment

---

## 简介

输入小红书笔记链接就能拉取评论数据，AI 自动帮你分析好评、吐槽、需求和竞品讨论，支持翻页浏览，还能导出精美 HTML 报告保存或分享。

**核心价值**

- **操作简单，粘贴即用**：给个笔记链接就行，不用记命令、不用学参数，直接出结果。
- **内容新鲜不滞后**：每次请求都从平台拉最新评论，不会拿老旧缓存糊弄你。
- **AI 帮你读懂评论**：自动识别好评、吐槽、用户需求和竞品讨论四类内容，提炼关键要点，不用自己一条条翻。
- **翻页浏览不封顶**：每页 20 条，cursor 游标翻页，想看多少翻多少，再多的评论也能看完。
- **精美报告可分享**：深色主题 HTML 报告，数据分析 + 原始评论一目了然，适合保存或转发给同事。

**适用对象**

- 🔍 小红书创作者 — 快速了解读者对笔记的真实反馈，找准内容优化方向。
- 📊 运营 / 数据分析 — 追踪评论区舆情动态，量化用户情绪变化。
- 🏢 品牌 / MCN — 监控竞品笔记评论风向，辅助投放与内容策略决策。
- 🛒 电商 / 种草 — 分析商品笔记评论中的用户需求与竞品讨论，优化种草话术。

---

## 功能特性

### 核心功能

- **链接提取**：支持直接粘贴小红书笔记链接，自动识别笔记 ID，无需手动提取。
- **评论获取**：获取笔记一级评论数据，每页 20 条按时间倒序展示。
- **AI 智能分析**：四维情感总结（积极评价 / 负面评价 / 用户需求 / 竞品对比），理解小红书社区文化。
- **分页浏览**：cursor 游标分页，支持逐页翻阅，大评论区也能完整浏览。
- **置顶标记**：置顶评论自动高亮标注，一眼识别重要声音。
- **可视化报告**：深色主题 HTML 报告，含原始评论表格与 AI 分析摘要，支持离线保存。
