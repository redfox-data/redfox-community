# 复刻蒸馏写作技能 / multi-copywrite-alchemy

---

## 简介

给你一个人的全部文章，还你一个**想得一样、写得一样**的写作分身。

不是让文章"像人写的"，是让文章"**像那个人写的**"——从思考方式到文字表达，全面复刻。

**核心价值**

- **思维级克隆**：不只学词汇句式，还提取作者的价值观、世界观和思维习惯，让分身的每篇文章都像本人在写
- **多平台自动采集**：支持小红书、公众号、抖音三个平台一键获取素材，无需手动复制粘贴
- **14种写作公式随心切换**：同一个分身，可以用PAS写痛点文、SCAR写故事、QUEST写教程，风格不变、思维不变

**适用对象**

- ✍️ **内容创作者** — 想复制喜欢的作者的写作能力，批量产出同风格内容
- 📊 **品牌 / 运营人员** — 需要统一文风的内容矩阵，高效产出
- 🔍 **风格研究者** — 想拆解某人的文章为什么有辨识度

---

## 功能特性

### 核心功能

- **🧠 双引擎蒸馏**：同时分析"他怎么想"和"他怎么写"，交叉验证后生成专属分身——换任何话题都像那个人在写
- **🔍 双重复检系统**：分身每次写作强制执行文风指纹复检 + 思想一致性复检，双检通过才输出
- **📝 14种写作公式**：内置PAS、QUEST、SCAR、AIDA等14种公式，风格不变、格式随心切换
- **📕📱🎵 多平台素材采集**：支持小红书、公众号、抖音自动获取文章，也可手动粘贴或提供文件路径
- **🔄 增量更新分身**：作者发了新文章？直接增量更新分身，不用从头再来

## 使用指南

直接用自然语言描述需求即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 平台采集蒸馏 | 「帮我从小红书账号XXX采集10篇文章进行蒸馏」 | 自动获取素材，分析后生成分身 |
| 手动粘贴蒸馏 | 「帮我分析这几篇文章的写作风格」（粘贴文章） | 直接分析提供的文本 |
| 使用分身写作 | 「用XX风格写一篇关于YY的文章」 | 调用分身，按该作者风格写作 |
| 更新已有分身 | 「XX最近有新文章，更新一下分身」 | 增量分析新文章，更新分身 |

### 素材要求

- 最少：≥3篇文章，总计≥2000字
- 理想：≥10篇文章，总计≥10000字
- 文章越多越准，10篇以上效果最佳
