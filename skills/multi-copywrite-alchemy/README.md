# 复刻蒸馏写作 / multi-copywrite-alchemy

---

## 简介

分析一位作者的多篇文章，深度还原其思维方式与写作风格，生成专属写作分身——从思考逻辑到文字表达，高度一致。

**核心价值**

- **全链路风格还原**：不只模仿词汇句式，而是从思维习惯到表达方式，让分身写出「那个人」才写得出的文章。
- **多平台素材接入**：支持公众号、抖音自动采集文章，也可直接粘贴或指定文件路径，灵活获取分析素材。
- **开箱即用的分身**：蒸馏完成后同步生成两份产出——可直接使用的风格提示词，以及可安装的本地分身子技能。

**适用对象**

- ✍️ **内容创作者** — 欣赏某位作者的表达，希望拥有同样的写作能力
- 📊 **运营人员** — 需要稳定产出同一风格的内容
- 🔍 **风格研究者** — 想拆解某位作者的文章为何具有高辨识度

---

## 功能特性

### 核心功能

- **深度风格还原**：不只分析表达习惯，还提取思维逻辑——生成一个从想法到文字都忠于原作者的专属分身
- **灵活写作模式**：分身支持多种写作框架，保持风格的同时适配不同文章类型
- **质量保障**：输出前自动校验风格一致性与内容准确性，确保品质稳定
- **多平台素材采集**：支持公众号、抖音直接获取文章素材，也可手动粘贴或指定文件路径

### 特色亮点

- **从思维到文字，全链路还原**：不是套模板换关键词，而是理解作者的思路后再创作——换话题、换写法，读起来仍是那个人
- **专属定制，非通用工具**：通用「去AI味」工具让文章像任何人写的，这个技能让文章像**那个人**写的

## 使用指南

直接用自然语言描述需求即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 平台采集蒸馏 | 「帮我从公众号XXX采集10篇文章进行蒸馏」 | 自动获取素材，分析后生成分身 |
| 手动粘贴蒸馏 | 「帮我分析这几篇文章的写作风格」（粘贴文章） | 直接分析提供的文本 |
| 使用分身写作 | 「用XX风格写一篇关于YY的文章」 | 调用分身，按该作者风格写作 |
| 更新分身 | 「XX最近有新文章，更新一下分身」 | 增量分析新文章，更新分身 |

### 输出示例

蒸馏完成后，你将获得：

- **思维+写作风格提示词**：可直接复制到任何AI对话中使用，涵盖作者的思维方式和写作铁律。
- **写作分身子技能**：已安装到本地，随时通过 `@作者名-style` 调用，支持指定写作模式。
