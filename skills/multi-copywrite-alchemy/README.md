# 复刻蒸馏写作技能 / multi-copywrite-alchemy

---

## 简介

基于一位作者的多篇文章，深度还原其思维方式与写作风格，生成专属写作分身。

区别于通用的风格模仿工具——分身不只是词汇句式相近，而是从观点立场、思维逻辑到行文节奏都与原作者保持一致。

**核心价值**

- **思维级还原**：不只学词汇句式，还提取作者的价值观与思维习惯，让分身写出的每篇都像本人在写
- **多平台自动采集**：小红书、公众号、抖音，输入账号即可获取文章素材，无需手动整理
- **14种写作公式灵活切换**：同一个分身，可写痛点文、故事、教程，风格不变、格式随心

**适用对象**

- ✍️ **内容创作者** — 欣赏某位作者的表达，希望复制其写作能力
- 📊 **品牌 / 运营人员** — 需要统一文风的内容矩阵，稳定高效产出
- 🔍 **风格研究者** — 想拆解某位作者的文章为何具有高辨识度

---

## 功能特性

### 核心功能

- **🧠 深度风格还原**：同时分析"怎么想"和"怎么写"，生成从思维到文字一致的专属分身
- **🔍 双重质量校验**：每次输出前自动完成文风匹配与思想一致性两轮检查，确保品质稳定
- **📝 14种写作公式**：内置PAS、QUEST、SCAR等14种写作框架，保持风格的同时灵活切换文章类型
- **📕📱🎵 多平台素材采集**：小红书、公众号、抖音直接获取文章，也可手动粘贴或指定文件路径
- **🔄 分身可增量更新**：作者发布新文章？直接更新分身，无需重新蒸馏

## 密钥获取与安全说明

- 本技能需要配置环境变量 `REDFOX_API_KEY`
- 前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，在 [密钥管理页](https://redfox.hk/settings/api-keys?source=github) 获取
- 将密钥配置到设备环境变量后即可使用
- 注意：不要在对话、代码或文件中明文暴露密钥

## 使用指南

直接用自然语言描述需求即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 平台采集蒸馏 | 「帮我从小红书账号XXX采集10篇文章进行蒸馏」 | 自动获取素材，分析后生成分身 |
| 手动粘贴蒸馏 | 「帮我分析这几篇文章的写作风格」（粘贴文章） | 直接分析提供的文本 |
| 使用分身写作 | 「用XX风格写一篇关于YY的文章」 | 调用分身，按该作者风格写作 |
| 更新已有分身 | 「XX最近有新文章，更新一下分身」 | 增量分析新文章，更新分身 |

### 素材要求

- 最少：≥3篇文章，总计≥2000字
- 理想：≥10篇文章，总计≥10000字
- 文章越多，还原越精准
