# 写作风格+思维框架 多维度分析框架

> 此框架用于系统性地分析一个作者的多篇文章，提取其独特的写作风格特征和思维框架。
> 基础7维（词汇→句法→篇章→修辞→情感→视角→语气）+ 格式适配分析（第8维）+ 思维框架分析（第9维）。
> 每个维度都要做到：**有数据支撑、有原文摘录、有可操作结论**。

---

## 维度一：词汇层 —— 用词的"指纹"

### 1.1 常用词与低频偏好
- **高频虚词**：统计"的、了、是、就、也、都、还、很"等虚词的出现频率和密度
- **实词偏好**：作者偏爱哪些实词？比如爱用"忧伤"而不是"伤心"、爱用"女人"而不是"女性"
- **口头禅/填充词**：文章中反复出现的口头禅式表达（如"说实话""你知道吗""我跟你说"）

**提取方法**：频次统计，列出 TOP 20 高频实词和 TOP 10 高频虚词，标注哪些词是作者特有的"指纹词"

### 1.2 词性分布
- **动词/名词/形容词的密度比例**
- 名词堆积多 → 偏静态描写、画面感强
- 动词强势 → 节奏快、叙事驱动
- 形容词密度高 → 情绪渲染型写作

**提取方法**：统计三类词的占比，判断主导词性及均衡度

### 1.3 词语搭配与习语
- 固定的主谓搭配（如"搂住夜色"而非"夜色降临"）
- 固定的动宾搭配（如"丢下一句话"而非"说了一句话"）
- 成语、俗语、文言词、外来词的使用频率和偏好

**提取方法**：标注10-15个作者的高频搭配，分析其来源（口语/书面/文学典故）

### 1.4 领域词与命名习惯
- 专业术语：作者是否有特定领域的词汇背景（法律、医学、商业等）
- 自造词：作者是否自创词汇或重新定义常见词
- 人名地名偏好：作者给角色取名有什么规律

**提取方法**：列出作者的领域词清单，标注自造词

## 维度二：句法层 —— 句子的"骨架"

### 2.1 平均句长与变化幅度
- **平均句长**（以字为单位）：短句多（<15字）显简洁锋利；长句多（>50字）显绵密深沉
- **句长标准差**：变化幅度大的作者有节奏感，句长恒定的作者风格稳定
- **极短句密度**：<5字的极短句使用频率，这是情绪爆发/强调的标志

**提取方法**：统计所有句子的长度，计算均值、中位数、标准差，标注句长分布特征

### 2.2 句式倾向
- 陈述句、疑问句、感叹句、祈使句的比例
- 反问句的使用频率（常用于跟读者建立亲密感）
- 设问句（自问自答）的使用频率
- 倒装、省略等特殊语序的使用

**提取方法**：按句末标点分类统计，标注特殊句式使用模式

### 2.3 从句与修饰结构
- 定语的长度与位置（前置修饰 vs 后置修饰）
- 状语的使用习惯（开头状语、插入状语、末尾状语）
- 层层嵌套的复杂从句 vs 简单句并列

**提取方法**：选取代表性段落做句式拆解，标注嵌套深度

### 2.4 连接方式
- **显性连接词**："但是""因此""然而""而且"等词的使用频率
- **意合连接**：省略连接词，靠语义自然衔接
- **标点用法**：分号、破折号、逗号的使用习惯（节奏控制手段）

**提取方法**：统计连接词密度（每千字个数），标点频率排序

## 维度三：篇章结构 —— 行文的"脉络"

### 3.1 开头模式
- 时间开头（"那年夏天""三年前的冬天"）
- 场景开头（直接描写一个场景画面）
- 观点断言开头（一句话亮观点）
- 对话开头（从一段对话切入）
- 引用开头（引用歌词/诗句/名言）

**提取方法**：分类统计所有文章的开头类型，标注主导类型

### 3.2 段落组织
- **段落长度习惯**：长段落（>300字）vs 短段落（<50字）vs 单句段落
- **段落节奏模式**：长-短-短-长 vs 均匀长度 vs 由长到短
- **段落间过渡**：跳跃式（缺乏过渡词）vs 承转衔接（用过渡句连接）
- **总分/分总结构偏好**：先给结论再展开 vs 先堆细节再归总

**提取方法**：统计段落长度分布，用典型文章的段落长度曲线可视化节奏

### 3.3 结尾风格
- 留白式（不说满，给读者想象空间）
- 总结式（提炼核心观点）
- 反转式（最后一段推翻前面的认知）
- 画面定格式（用一个画面收尾）
- 行动指令式（"你怎么看？""留言告诉我"）

**提取方法**：分类统计所有文章的结尾类型

### 3.4 逻辑推进方式
- 顺叙（按时间推进）
- 倒叙（先讲结果再回溯）
- 插叙（中间插入回忆/侧面信息）
- 因果链论证
- 类比推理
- 事例堆叠

**提取方法**：分析每篇文章的整体推进方式，标注主导模式

## 维度四：修辞与表达方式 —— 文字的"颜色"

### 4.1 修辞格偏好
- **比喻**：使用密度、喻体来源偏好（自然景物？日常生活？科技？）
- **拟人**：是否惯于给非人事物赋予人的行为/情感
- **排比**：是否经常使用排比句式（三段式排比尤其注意）
- **反复**：是否有意识地重复关键词/句式
- **对比/对照**：是否经常设置对立面
- **夸张**：使用频率和夸张程度

**提取方法**：统计每种修辞格的出现频次，标注2-3个最典型的使用案例

### 4.2 引用与用典
- 引用歌词、电影台词、诗句、名人名言
- 引用方式：直接引（带引号）vs 化用（融为自己的语言）
- 引用来源：偏好什么类型的引用（流行文化？古典文学？网络梗？）

**提取方法**：列出作者引用的来源清单，分析引用偏好

### 4.3 具象/抽象层次
- **细节描写的颗粒度**："一片枯黄的梧桐叶"（高颗粒度）vs "秋意"（低颗粒度）
- **感官调用侧重**：视觉、听觉、嗅觉、触觉、味觉，作者偏好哪种？
- **具象词 vs 抽象词比例**：偏画面感还是偏概念论述

**提取方法**：统计具象名词和抽象名词的比例，标注感官描写类型分布

## 维度五：情感与态度 —— 文章的"温度"

### 5.1 情感极性分布
- 整体偏向：积极 / 消极 / 矛盾复杂 / 平淡克制
- 情绪浓淡：热烈外放 vs 含蓄克制
- 情感词汇密度：每千字情感词数量

**提取方法**：统计正面/负面/中性情感词比例，绘制情感分布图

### 5.2 态度与立场
- 批判型（对社会/他人持批判态度）
- 赞美型（倾向于发现和赞美美好）
- 讽刺型（用反讽表达不满）
- 自嘲型（拿自己开涮）
- 怀念型（沉浸在过去的情绪中）
- 冷静旁观型（保持距离，不轻易表态）
- 主观程度词密度："也许/一定/简直/绝对/可能"等词的使用频率

**提取方法**：对每篇文章做态度标注，统计主导态度类型

### 5.3 情绪节奏
- 高开低走（开头情绪强，结尾平淡）
- 逐步升温（从平淡到高潮）
- 平静中突然爆发（积蓄后释放）
- 波浪式（情绪起伏有规律）

**提取方法**：选取代表性文章画情绪曲线（段落级情感强度变化）

## 维度六：内容倾向与视角 —— 作者"看世界"的方式

### 6.1 高频题材/母题
- 常写什么主题？故乡、孤独、成长、爱情、婚姻、职场、日常琐碎、宏大叙事
- 常用意象：月光、车站、镜子、酒、雨、黄昏、老照片……
- 话题范围：只写个人经历？还是涉猎广泛？

**提取方法**：对所有文章做主题标注，统计高频题材和常用意象 TOP 10

### 6.2 叙事视角与人称
- 第一人称（我）：内心独白式 vs 故事讲述式
- 第二人称（你）：直接对话读者
- 第三人称（他/她）：全知视角 vs 有限视角
- 视角切换频率：始终如一 vs 经常跳转

**提取方法**：统计各人称代词密度，标注叙事视角类型

### 6.3 价值观与信念词
- 反复出现的抽象词：自由、爱、规则、理想、活着、体面、尊严……
- 对这些词的情感倾向：正面拥抱 vs 质疑解构 vs 矛盾纠结
- 是否有明确的道德立场或价值观输出

**提取方法**：词频分析找出高频抽象词，结合上下文做情感倾向判断

## 维度七：语气与节奏 —— 读起来的"感觉"

### 7.1 口语化/书面化程度
- 语气词密度："呢、啊、吧、吗、嘛、啦、呀"等
- 儿化音：北方作者多见
- 口语表达：是否使用"就是说""那种""什么的"等口语化结构
- 书面表达：是否偏文绉绉，用词典雅

**提取方法**：统计语气词密度，判断口语化指数（1-10分）

### 7.2 节奏快慢
- **快节奏**：短句多、连动多、标点少、信息密度高
- **慢节奏**：长句多、修饰多、停顿多、情绪铺陈
- **节奏变化模式**：何时加速？何时减速？

**提取方法**：通过句长变化和标点密度分析节奏模式

### 7.3 留白与信息密度
- **高信息密度**：事无巨细全部写出，不给读者想象空间
- **高留白**：擅长只给意象，让读者自己脑补
- **词汇多样性**：用词丰富还是重复使用核心词汇
- **信息熵**：文本的不可预测性，越高说明越丰富多样

**提取方法**：计算词汇多样性（type-token ratio），分析每段的信息负载

## 维度八：格式适配分析 —— 作者的"最佳战场"

> 基于前7维的分析结果，判断作者最适合哪种写作公式。
> 目标是告诉使用者：这个人写什么类型的文章最好看。

### 8.1 结构惯性分析

从维度三（篇章结构）提取作者的**默认组织方式**：

| 作者惯用结构 | 自然匹配的公式 | 匹配理由 |
|-------------|--------------|---------|
| 先抛问题再解答 | PAS、QUEST | 作者的"问题-解答"思维惯性 |
| 按时间顺序叙述 | SCAR | 故事叙述是本能 |
| 层层递进推结论 | 情绪曲线公式 | 擅长控制读者情绪节奏 |
| 对比展开 | BAB | 对比是思维方式 |
| 列举+评测 | 列表体2.0、FAB | 作者天然就是"测评人" |

### 8.2 能力雷达图匹配

将作者的7维特征映射到各类公式所需的能力：

| 公式 | 需要的能力 | 匹配判断 |
|------|-----------|---------|
| **PAS** | 精准定义问题的能力、痛点共情力、解法清晰度 | 作者是否擅长提炼问题？是否有强共情力？ |
| **QUEST** | 共情开场力、系统性教学能力、行动号召力 | 作者是否有"老师"气质？教学是否清晰？ |
| **SCAR** | 故事叙述力、冲突设计力、顿悟提炼力 | 作者是否天然就是说故事的人？故事是否打动人心？ |
| **AIDA** | 抓注意力、制造欲望、推动行动 | 作者是否有"销售"气质？文字是否有说服力？ |
| **情绪曲线** | 情绪操控力、节奏控制力 | 作者是否擅长长篇情绪铺陈？ |
| **列表体2.0** | 系统测评力、判断标准力 | 作者是否有"筛选者"气质？ |
| **爆款笔记四步** | 痛点共情+干货密度+金句力 | 作者的小红书属性有多强？ |
| **Hook-Stack-CTA** | 一句话抓人力、信息密度 | 作者的"一句话"能力有多强？ |

### 8.3 推荐输出

基于以上分析，输出：

```
格式适配推荐

天生适合（⭐⭐⭐）：
  - [公式1]：[理由，基于哪些维度特征]
  - [公式2]：[理由]

可以驾驭（⭐⭐）：
  - [公式3]：[理由]
  - [公式4]：[理由，需要哪些微调]

不太适合（⭐）：
  - [公式X]：[为什么不建议，会丢失作者的哪些核心优势]

首推写作模式：[最推荐的一个公式]
  选这个模式写的第一篇文章应该是：[具体选题建议]
```

### 8.4 公式适配规则总结

基于7维分析+格式适配分析，总结该作者**适配任意公式**时应遵循的调整规则：

```
通用适配规则：
- 当使用故事型公式（SCAR/情绪曲线）时：[放大作者的X特征，收敛Y特征]
- 当使用教学型公式（QUEST/PAS）时：[调整X，保留Y]
- 当使用销售型公式（AIDA/4Ps/BAB）时：[注意X陷阱]
- 当使用短内容公式（三段式/Hook-Stack-CTA）时：[压缩X，强化Y]
- 当使用平台专用公式时：[保留核心风格，适配平台约束]
```

## 维度九：思维框架分析 —— 作者的"操作系统"

> 从文章中反向提取作者的思维框架——他的价值观、世界观、核心信念、决策模式。
> 核心假设：**一个人的思维框架，一定会在他的写作中泄露出来。**
> 区别女娲：女娲通过全网搜索（访谈、书籍、社交媒体）蒸馏思维框架，本维度通过文章的文本分析蒸馏思维框架。文章思维框架 = 写作中暴露的思维方式，不一定等于作者真实生活中的全部思维方式，但一定等于他写作时的思维方式。

### 9.1 价值观体系

从文章中提取作者"觉得什么重要"：

| 分析项 | 提取方法 |
|--------|---------|
| **高频价值观词** | 统计"自由、爱、规则、理想、活着、体面、尊严、选择、公平、善良、强大、温柔"等抽象价值观词的频率和上下文情感倾向 |
| **赞美对象** | 作者赞美什么样的人/事？列举3-5个典型被赞美对象 |
| **批判对象** | 作者批判什么样的人/事？列举3-5个典型被批判对象 |
| **价值排序** | 从文章冲突场景中推断价值优先级（如：面子 vs 里子，亲情 vs 规则） |

**输出格式**：

```
价值观体系
├── 核心价值观（3-5个）：[价值词] - [文章中如何体现]
├── 赞美对象：... 
├── 批判对象：...
└── 价值冲突时的排序：[X] > [Y] > [Z]（从文章中冲突场景推断）
```

### 9.2 世界观与核心信念

从文章中提取作者"怎么看待这个世界"：

| 分析项 | 提取方法 |
|--------|---------|
| **世界基调** | 作者笔下的世界是温暖的/冷酷的/荒诞的/复杂的？统计正面/负面事件在文章中的比例和描述方式 |
| **人性假设** | 作者默认"人本善"还是"人本恶"？看他对陌生人、家人、同事的默认描写倾向 |
| **核心信念** | 反复出现≥3次的信念式表述（如"亲情是最靠不住的""钱能解决99%的问题"） |
| **因果关系认知** | 作者认为事情的结果由什么决定？努力？运气？出身？社会？关系？ |

**输出格式**：

```
世界观画像
├── 世界基调：[一句话概括]
├── 人性假设：[默认信任/默认怀疑/视情况而定]
├── 核心信念（3-5条）：[信念] - [出现次数] - [典型出处]
└── 因果归因：[努力/运气/出身/社会/关系] ← 主导因素
```

### 9.3 身份认同与立场

从文章中提取作者"觉得自己是谁"：

| 分析项 | 提取方法 |
|--------|---------|
| **自我定位** | 作者在文章中把自己放在什么位置？导师/朋友/同行者/记录者/控诉者？ |
| **群体归属** | 作者频繁使用"我们"指代什么群体？（"我们这代人""我们女人""我们打工人"） |
| **立场选择** | 在争议话题中，作者站哪边？从叙事视角和情感色彩判断 |
| **同理心半径** | 作者对哪些群体有同理心？（家人？同龄人？陌生人？对手？）对哪些群体不共情？ |

**输出格式**：

```
身份认同画像
├── 自我定位：[角色] - [为什么]
├── 群体归属：作者口中的"我们"= [群体描述]
├── 立场倾向：[保守/变革/中立/矛盾]
└── 同理心半径：内圈=[X]，中圈=[Y]，外圈=[Z]，盲区=[W]
```

### 9.4 思维习惯与论证方式

从文章中提取作者"怎么想问题"：

| 分析项 | 提取方法 |
|--------|---------|
| **论证方式** | 作者偏好什么论证？举例论证/类比论证/因果论证/权威引用/情感诉诸？ |
| **思维工具** | 作者习惯用什么思考工具？对比分析/二分化/层层追问/抽象升华/具体还原？ |
| **矛盾处理** | 当遇到矛盾时，作者怎么处理？承认&分析/回避/用情感覆盖/强行融合？ |
| **确定程度** | 作者有多确定自己的观点？"一定是""肯定是"型 vs "可能""也许""不知道"型 |

**输出格式**：

```
思维习惯画像
├── 主导论证方式：[方式] - [占比]
├── 常用思维工具：[工具1]、[工具2]、[工具3]
├── 矛盾处理模式：[承认分析/回避/情感覆盖/强行融合]
└── 确定程度：[高度确定型/开放探索型/情境可变型]
```

### 9.5 对立面与禁忌

从文章中提取作者"绝对不接受什么"：

| 分析项 | 提取方法 |
|--------|---------|
| **情绪触发点** | 什么事情会让作者明显变得愤怒/激烈/讽刺？统计情绪强度突变的段落 |
| **不可妥协** | 作者明确说"不能接受""底线""绝不"的事情 |
| **沉默地带** | 作者刻意回避不谈的话题？从文章题材的排除中推断 |
| **思维盲区** | 作者可能没意识到但读者能察觉的认知局限？从反复出现的偏见模式中推断 |

**输出格式**：

```
对立面与禁忌
├── 情绪触发点（3-5个）：[触发点] - [文章中的典型反应]
├── 不可妥协的底线（2-3条）：[底线] - [为什么]
└── 盲区提示：[可能存在的认知局限] - [证据]
```

### 9.6 思维画像总结

综合9.1-9.5，用一段话总结此人的思维画像：

## 双引擎交叉验证：思维基因 → 写作基因的映射

> 这是 v3.0 最关键的创新。完成维度九（思维分析）和维度一~八（写作分析）后，执行**跨引擎交叉验证**——找出思维框架如何驱动写作风格。

### 交叉验证逻辑

思维画像中的每个特征，寻找它在写作风格中的"物证"：

| 思维特征（维度九） | → | 写作物证（维度一~八） |
|-------------------|---|---------------------|
| 批判型态度 | → | 高频反问句（维度四）、负面情感词>正面（维度五） |
| 人性默认不信任 | → | 故事中人物总是被背叛（维度六）、极少直白的信任表达（维度五） |
| 看重"体面" | → | 高频出现"体面""丢人""难看"（维度一）、故事冲突围绕面子展开（维度六） |
| 用故事思维而非理论思维 | → | 抽象词密度低、具象词密度高（维度四）、开头总是具体场景（维度三） |
| 确定性高/不犹豫 | → | 短句比例高、极少"也许/可能"（维度一）、态度词密度高（维度五） |

### 交叉验证输出：思维基因

从交叉验证中提炼3-5条 **"思维基因"**——最底层的、同时驱动思维和写作的核心特征：

```
思维基因 #1：[一句话概括] 
  → 思维层面：[它如何影响作者怎么想]
  → 写作层面：[它如何影响作者怎么写]
  → 反例测试：如果作者在想/写时违背了这条基因，会[产生什么违和感]

思维基因 #2：...
```

**思维基因 vs 写作基因的区别**：
- 写作基因：描述的是"他怎么写"（如"语言极简，一句一顿"）
- 思维基因：描述的是"他为什么这样写"（如"他觉得啰嗦是虚伪的表现，所以他写东西一句一顿"）
- 思维基因是根，写作基因是叶。理解思维基因，才能在写任何类型的文章时保持"像他"。

## 综合提炼：写作基因

完成7+1维分析后，提炼3-5个作者的"写作基因"——即最能代表该作者风格、一旦改变就不像他的核心特征。

格式：
```
写作基因 #1：[一句话概括] - [来自哪几个维度的交叉印证]
写作基因 #2：[一句话概括] - [来自哪几个维度的交叉印证]
...
```

这些写作基因将作为生成提示词和技能时最核心的约束条件。
