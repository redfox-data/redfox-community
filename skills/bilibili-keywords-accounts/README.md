# B站关键词搜账号 / bilibili-keywords-accounts

---

## 简介

输入关键词即可搜索B站相关账号，获取含粉丝数、等级、简介等信息的账号列表，支持按粉丝数排序与智能分类，帮你快速发现和筛选目标领域的UP主。

**核心价值**

- **关键词驱动**：输入 2~6 字关键词，快速获取B站相关领域账号列表，无需逐个手动搜索。
- **智能分类**：自动按内容方向（游戏、美食、科技等）归类账号，结构化浏览更高效。
- **多维信息**：一次返回粉丝数、等级、简介、直播状态等核心数据，快速评估账号基本面。

**适用对象**

- 🔍 **内容运营 / MCN** — 快速发现同赛道UP主，拓展合作资源与对标账号。
- 📊 **品牌方 / 广告主** — 按领域筛选KOL，评估粉丝量级与账号活跃度。
- 🎯 **创作者 / 研究者** — 了解B站某领域的账号生态与内容分布。

---

## 功能特性

### 核心功能

- **关键词搜索**：根据关键词搜索B站账号，快速发现目标领域UP主。
- **多维排序**：支持综合排序与按粉丝数排序，按不同维度筛选优质账号。
- **账号详情**：返回粉丝数、等级、简介、直播状态等核心信息。
- **AI 智能分类**：自动按内容方向归类账号（游戏、美食、科技等 12 大分类），结构化浏览更高效。
- **分页浏览**：支持翻页查看更多账号，扩大筛选范围。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述你想搜索的B站账号方向或关键词即可，无需记忆固定命令。

### 常用说法速查

| 意图           | 示例话术                                 | 效果                                       |
| -------------- | ---------------------------------------- | ------------------------------------------ |
| 搜索某领域账号 | 「帮我搜一下B站美食相关的账号」           | 按关键词搜索，自动分类展示账号列表         |
| 找同类型UP主   | 「B站做科技测评的账号有哪些」             | 返回科技类账号，按粉丝数排序               |
| 翻页查看更多   | 「查看更多」                             | 自动翻到下一页，继续展示更多账号           |
| 宽泛搜索       | 「搜一下游戏账号」                       | 返回游戏大类下的账号，自动细分子方向       |
