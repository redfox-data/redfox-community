# 小红书爆款封面生成

## 简介

小红书爆款封面生成是一款专为小红书内容创作者打造的 AI 封面设计工具，基于全网每日持续收录的 **2000+ 条爆款笔记数据**，深度抓取同赛道爆款封面的视觉元素，通过 AI 智能分析总结高转化视觉规律，生成贴合笔记内容、符合平台流量审美的可落地封面设计方案。

通过简单的主题描述，你可以：
- 🔍 自动解析创作意图，精准提取关键信息
- 📊 获取目标领域真实爆款封面图，AI 分析总结风格特征
- 🎨 结合用户需求与爆款数据，生成可落地的设计方案（含案例 + 生图提示词）

适用于小红书博主、内容运营、品牌方、MCN 机构等需要提升笔记封面点击率的场景。

---
  
## 功能特性

### 🎯 核心功能
- **🧠 智能意图解析**：自动解析用户文案意图，提取内容主题、类型、风格偏好和核心关键词
- **📊 爆款数据查询**：调用红狐数据接口，获取低粉高赞、点赞最多、单日互动爆发、7日持续增长四维爆款数据
- **🖼️ AI 图像分析**：逐张识别爆款封面图的文字、场景、色调、构图等视觉元素，分类汇总风格规律
- **🎨 定制方案生成**：结合用户需求与爆款风格特征，输出三种差异化封面设计方案（含生图提示词）

### ✨ 特色亮点
- **📈 数据驱动**：基于日更 2000+ 条真实爆款笔记数据，拒绝拍脑袋设计
- **⚡ 全流程自动化**：从意图解析到方案生成，7 步标准流程一键完成
- **🎯 精准匹配**：智能评分筛选机制，确保推荐内容与用户需求高度相关
- **📱 平台适配**：所有方案均适配小红书 3:4 竖版封面规格（1080x1440像素）

## 一键安装

### 前置条件
- Python 3.8+ 运行环境
- 已获取红狐数据 API Key

### 获取 API Key
1. 访问 [红狐Hub 官网](https://redfox.hk/) 了解服务详情
2. 前往 [注册页面](https://redfox.hk/login) 注册账号
3. **新注册用户将获赠免费积分**，可立即开始使用 API 服务
4. 注册登录后，在个人中心获取 API Key，格式为 `ak_xxxxxxxx`

### 配置 API Key
`REDFOX_API_KEY` 从环境变量获取，格式 `ak_xxxxxxxx`

若未设置，提示用户自行配置；若用户不会配置，Agent 应主动帮用户设置：

| 操作系统 | 配置命令 |
|----------|----------|
| macOS/Linux（zsh） | 将 `export REDFOX_API_KEY=<值>` 追加到 `~/.zshrc`，然后 `source ~/.zshrc` |
| macOS/Linux（bash） | 将 `export REDFOX_API_KEY=<值>` 追加到 `~/.bashrc`，然后 `source ~/.bashrc` |
| Windows | 执行 `[Environment]::SetEnvironmentVariable("REDFOX_API_KEY", "<值>", "User")`（需重启终端生效） |

配置完成后应验证：`echo $REDFOX_API_KEY`（macOS/Linux）或 `echo %REDFOX_API_KEY%`（Windows），确保换一个 skill 也能读取到

### 依赖安装
```
pip install requests>=2.28.0
```

## 使用指南

### 基础使用

#### 1. 描述你的封面需求
告诉助手你想设计的封面主题：

> 用户：帮我设计一个穿搭类封面
> 
> 助手：好的！我将为您分析穿搭赛道的爆款封面数据，生成专属封面设计方案。

#### 2. 等待分析完成
助手将自动执行7步工作流程（意图解析 → 数据查询 → 智能筛选 → 图像分析 → 风格总结 → 方案生成 → 自检），全程约1-2分钟。

#### 3. 选择喜欢的方案
查看三种不同风格的封面方案，获取生图提示词，直接用于封面制作。

### 高级使用

#### 自定义时间范围
> 用户：帮我分析近7天的穿搭爆款封面

#### 指定细分赛道
> 用户：设计一个小个子通勤穿搭的封面，风格偏韩系

#### 上传图片结合分析
> 用户：[上传图片] 结合我的照片和爆款方案生成专属封面

### 完整工作流程

核心工作流程包含 **7个标准步骤**，详细执行规范请参考 [references/core_workflow.md](references/core_workflow.md)：

| 步骤 | 名称 | 说明 |
|------|------|------|
| 步骤1 | 用户意图解析 | 结构化解析需求，提取主题/类型/风格/关键词 |
| 步骤2 | 爆款数据查询 | 调用脚本获取四维爆款数据 |
| 步骤3 | 智能匹配筛选 | 综合评分筛选20条最相关数据 |
| 步骤4 | AI图像分析 | 逐张识别封面视觉元素，分类汇总 |
| 步骤5 | 风格总结 | 输出爆款封面风格总结表（含真实图片） |
| 步骤6 | 方案生成 | 输出三种差异化封面设计方案 |
| 步骤7 | 输出自检 | 逐项检查输出完整性 |

> ⚠️ **重要**：执行任务时，Agent 必须严格按照 [core_workflow.md](references/core_workflow.md) 中定义的流程逐步执行，不可跳过任何步骤。

## 常见问答

### 安装相关问题

**Q1: 提示"未找到 REDFOX_API_KEY 环境变量"怎么办？**

A: 请按以下步骤配置：
1. 访问 [红狐Hub](https://redfox.hk/) 注册账号并获取 API Key
2. Windows 用户：在 PowerShell 中执行 `[Environment]::SetEnvironmentVariable("REDFOX_API_KEY", "你的API Key", "User")`，然后重启终端
3. macOS/Linux 用户：将 `export REDFOX_API_KEY=你的API Key` 追加到 shell 配置文件，然后 `source` 使其生效
4. 验证配置：执行 `echo %REDFOX_API_KEY%`（Windows）或 `echo $REDFOX_API_KEY`（macOS/Linux）

**Q2: 支持哪些平台部署？**

A: 本技能为 Agent Skill 格式，支持所有兼容 SKILL.md 规范的 AI Agent 平台。

---

### 使用相关问题

**Q3: 查询的数据时间范围是多久？**

A: 数据库仅包含昨天至30天前的数据，默认查询近30天。如需自定义时间范围，可在请求时指定"近N天"。

**Q4: 可以同时查询多个关键词吗？**

A: 支持。最多5个关键词，用逗号分隔（如：`职场穿搭,通勤穿搭,面试穿搭`），总长度不超过200字符。

**Q5: 生成的封面方案可以直接使用吗？**

A: 每个方案均包含生图提示词，可直接用于 AI 绘图工具生成封面图。提示词已包含 3:4 竖版比例（1080x1440像素）要求。

---

### 故障排除

**Q6: API 调用失败怎么办？**

A: 请按以下步骤排查：
1. 确认 API Key 是否正确配置（参考 Q1）
2. 确认网络连接正常
3. 脚本内置3次自动重试，如仍失败请稍后再试
4. 确认 API Key 是否仍有余额

**Q7: 返回数据为空怎么办？**

A: 可能原因：
- 关键词过于小众或新兴，爆款内容积累较少
- 近期该赛道热度较低
- 建议更换更热门的关键词，如"早八穿搭"、"减脂餐"等
