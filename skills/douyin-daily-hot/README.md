# 抖音每日热门作品榜 / douyin-daily-hot

---

## 简介

日度收录全平台抖音作品，输出单日点赞 TOP50 榜单，支持按赛道分类查询与历史日期回溯。

**核心价值**

- **每日收录核验**：每日人工收录核验全平台万级抖音作品，榜单质量有保障。
- **赛道全覆盖**：支持 28 个主流赛道分类查询，从美食到游戏一键直达。
- **历史可回溯**：最多回溯过去 30 天数据，轻松复盘热度趋势。

**适用对象**

- 📊 **内容运营** — 快速获取各赛道热门内容，把握平台热度风向。
- 🎬 **创作者 / 博主** — 了解同赛道爆款表现，对标热门作品优化创作。
- 📈 **品牌 / MCN** — 监测竞品账号与赛道排名，优化投放与内容策略。

---

## 功能特性

### 核心功能

- **日榜排名**：每日输出单日点赞 TOP50 榜单，快速获取全平台当日最热作品。
- **赛道筛选**：支持 28 个赛道分类查询，精准定位垂直领域热门内容。
- **历史回溯**：支持回溯过去 30 天数据，追溯往期热门趋势。
- **可点击链接**：作品标题以超链接格式输出，一键跳转查看原作品。
- **分页展示**：默认 TOP20，可查看完整 TOP50，按需获取全量或摘要数据。
- **订阅推送**：支持赛道和时间偏好订阅，定时获取热门动态。

## 使用指南

直接用自然语言描述需求即可，无需记忆固定命令。

### 常用说法速查

| 意图         | 示例话术                     | 效果                           |
| ------------ | ---------------------------- | ------------------------------ |
| 查看今日榜单 | 「抖音热榜」「今日榜单」     | 返回昨日全品类点赞 TOP20       |
| 按赛道查看   | 「看看美食赛道」「二次元热门」 | 返回对应赛道点赞 TOP20         |
| 查看历史     | 「5月20号的榜单」            | 返回指定日期的榜单             |
| 查看完整数据 | 「查看全部」「看剩余30条」   | 展示完整 50 条数据             |

### 输出示例

查询后将以 Markdown 表格呈现，默认展示 TOP20：

| 排名 | 作品标题               | 作者（粉丝数） | 赛道 | 收藏 | 评论 | 分享 | **点赞** | 发布时间 |
| ---- | ---------------------- | -------------- | ---- | ---- | ---- | ---- | -------- | -------- |
| 1    | [红烧肉的教程](链接)   | 美食达人（10w+） | 美食 | 10w+ | 10w+ | 10w+ | **10w+** | 05-28 12:00 |

> 查询特定赛道时不含「赛道」列；作品标题可直接点击跳转抖音原作品。

## 重要数据说明

- 每日 **06:00** 更新昨日数据（数据为 T-1，即最新可用数据为昨日）。
- 支持回溯过去 **30 天**的历史数据。
- 互动数据为入库时间快照，与实时数据存在差异。
- 每个赛道最多返回 **50 条**数据。
