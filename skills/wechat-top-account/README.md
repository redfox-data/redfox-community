# 公众号热门账号推荐 / wechat-top-account

---

## 简介

快速获取微信公众号综合实力排行榜 TOP50，支持日榜/周榜/月榜及 23 个垂直领域筛选，展示综合评分与多维互动指标，可导出 HTML 可视化报告并订阅定期推送。

**核心价值**

- 覆盖 23 个垂直分类，日/周/月三档榜单随时切换，一次查询掌握赛道头部格局
- 综合评分基于阅读、点赞、在看、转发多维加权，结果客观可量化
- 榜单之外可进一步获得内容生态、热点影响力、蓝海赛道三类深度解读

**适用对象**

- 📱 公众号运营者 — 了解所属赛道头部账号的内容产出与互动表现，明确对标方向
- 🏢 品牌营销团队 — 跟踪竞品排名变化，及时调整内容策略
- 📊 MCN 机构 — 订阅定期推送，导出可视化报告支撑复盘与客户汇报

---

## 功能特性

### 核心功能

- **多维排行查询**：按日榜、周榜、月榜查询 23 个分类 TOP50，输入关键词自动匹配分类
- **综合评分体系**：基于阅读、点赞、在看、转发等指标计算综合评分，满分 100
- **深度分析解读**：提供内容生态变化、热点影响力量化、蓝海赛道挖掘三种解读视角
- **HTML 可视化报告**：生成可视化榜单页面，支持导出图片和 PDF
- **定时订阅推送**：订阅日/周/月榜，按更新时间自动推送最新数据

### 特色亮点

- 日榜每日 17:30、周榜每周一 17:30、月榜每月 3 日 23:00 自动刷新，数据及时
- 榜单表格与 HTML 报告共用同一数据源，顺序与内容完全一致
- 日榜可回溯近 7 天、周榜近 3 周、月榜近 3 个月，超出范围自动调整并提示

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 查分类日榜 | `查询科技类公众号日榜` | 输出科技数码分类日榜 TOP50 表格，含综合评分与互动指标 |
| 查分类周榜 | `本周美食类公众号排名` | 输出美食餐饮周榜 TOP50，标注数据时间周期 |
| 获取深度分析 | 榜单输出后回复 `1` | 基于 TOP50 数据输出内容生态、热点影响、蓝海赛道三类分析 |
| 生成 HTML 报告 | 榜单输出后回复 `2` | 生成可导出图片/PDF 的可视化榜单页面 |
| 订阅定期推送 | `订阅周榜` / `订阅日榜` | 按对应更新时间自动推送最新榜单 |
| 列出全部分类 | `有哪些分类可以查？` | 列出全部 23 个可用垂直分类 |

### 输出示例

榜单以表格形式直接输出，每条包含排名、账号名称（可跳转）、综合评分、发布数、总阅读、头条阅读、最高阅读、总点赞、总在看、总转发。输出完成后附功能引导，按需选择分析、报告或订阅。
