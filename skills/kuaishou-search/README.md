# 快手作品搜索 / kuaishou-search

---

## 简介

输入关键词就能搜到快手热门作品，支持按排序方式、发布时间多维度筛选，作品标题、作者、播放数、点赞数、评论数、收藏数一览无余，点一下链接就能跳转看详情。

**核心价值**

- **即搜即看**：输入任意内容关键词，立刻获取按点赞数排序的爆款作品列表。
- **数据全面**：每条作品展示播放数、点赞数、评论数、收藏数及可跳转的作品链接，发布时间精确到时分秒。
- **多维筛选**：支持按播放量、点赞、评论、收藏等多种维度排序，灵活切换视角发现好内容。
- **持续关注**：支持订阅感兴趣的关键词，每天定时推送最新作品数据，不错过任何趋势。

**适用对象**

- 🔍 **内容创作者** — 追踪同赛道爆款，为选题与创作提供数据参考。
- 📊 **运营 / 数据分析** — 快速了解某个品类在快手上的热门内容，掌握流量趋势。
- 🏢 **品牌 / MCN** — 筛选潜在合作达人，按互动量评估作品价值，辅助投放决策。
- 🛒 **电商选品** — 搜索细分赛道的带货作品，找到匹配的推广内容方向。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入关键词搜索快手热门作品，精准发现目标赛道内容。
- **多维排序**：支持综合排序、最新发布、最多点赞、最多播放、最多评论、最多收藏六种排序方式。
- **时间筛选**：支持不限、一天内、一周内、一个月内、一年内五种时间范围。
- **分页浏览**：每页展示 20 条结果，支持多页翻看，不遗漏任何内容。
- **可点击链接**：作品标题以超链接格式输出，一键跳转查看原作品。
- **订阅推送**：支持关键词每日推送，定时获取最新作品动态。

## 使用指南

直接用自然语言说出你想查的内容方向即可，无需记忆固定命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 搜索某类内容 | "搜一下最近一周的美食内容" | 以"美食"查询一周内作品，按点赞排序 |
| 指定排序方式 | "看看最近播放最多的世界杯作品" | 以"世界杯"查询，按播放量排序 |
| 指定时间范围 | "搜索六月以来热门的旅游内容" | 解析日期范围，返回该时段热门作品 |
| 切换排序条件 | "按最新发布、一周内重新搜索" | 切换排序为最新发布，时间为一周内 |
| 翻页查看更多 | "下一页" | 自动递增页码，展示后续 20 条作品 |
| 订阅每日推送 | "确认订阅" | 每天 09:00 推送最新发布的不限时间作品 |

## 重要数据说明

- 数据每日早上 **06:00** 更新，并非实时查询。
- 订阅推送每天 **09:00** 执行，推送内容按最新发布排序、不限时间范围。
- 数字 ≥ 10000 时以 `x.xw` 格式展示（如 `1.2w`），便于快速阅读。
