# 抖音实时作品搜索 / douyin-realtime-search

---

## 简介

输入任意关键词，实时拉取抖音平台当下最新作品数据，告别历史缓存、看见真正的现在。

**核心价值**

- 实时获取：直连抖音搜索服务，结果反映当前平台最新状态，而非历史收录数据
- 灵活筛选：支持按排序方式（综合 / 最新发布 / 最多点赞）与发布时间（7天 / 30天 / 90天 / 不限）自由组合
- 结构化呈现：作品标题、作者、点赞、评论、分享、收藏数一览无余，可直接点击跳转原作品

**适用对象**

- 📊 **内容运营 / 策略研究者** — 快速掌握某赛道当前真实热度与新发趋势
- 🎬 **创作者 / 博主** — 实时了解同类内容最新动态，找准发布时机

---

## 功能特性

### 核心功能

- **关键词实时搜索**：输入细分方向词，直接拉取该关键词在抖音平台的实时作品列表
- **多维筛选组合**：排序方式与发布时间可自由组合，精准控制搜索范围
- **结果全量展示**：搜索结果一次性全部展示，无需翻页
- **筛选状态透明**：每次输出后告知当前生效的排序与时间参数，并提示切换方式
- **每日订阅推送**：可订阅关键词，每天 10:00 自动推送最新搜索结果

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`) 提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 搜索某关键词 | 「健身」/ 「职场穿搭」 | 按综合排序、最近7天实时搜索并展示结果 |
| 指定排序方式 | 「按最新发布搜索减脂」 | 切换为最新发布排序后搜索 |
| 指定时间范围 | 「搜索近30天的AIGC内容」 | 将发布时间筛选设为最近30天 |
| 组合筛选 | 「按最多点赞、最近90天搜索古法健身」 | 同时指定排序和时间后搜索 |
| 切换筛选条件 | 「按最新发布、最近30天重新搜索」 | 保留关键词，更换筛选参数后重新拉取 |
| 订阅每日推送 | 「确认订阅」 | 创建定时任务，每天10:00自动推送 |
