# 微博热搜 / weibo-hot-search

---

## 简介

打开就能看到微博当前热搜榜，热搜标题和排名一目了然，还能帮你归纳热搜背后的创作灵感和选题方向，点一下就能跳转查看相关博文，给你的是平台刚更新的榜单，不会拿旧数据糊弄你。

**核心价值**

- **榜单新鲜不滞后**：每次打开直接拉取微博当前热搜，只给你最新的榜单排名。
- **排名清晰直观**：按热搜排名从 1 到 N 逐条展示，热度高低一目了然。
- **一键直达搜索**：每条热搜附带微博搜索链接，点击即可查看相关博文和讨论。
- **灵感分析总结**：自动归纳热搜赛道分布，提炼创作选题建议和风险提示，帮你找到内容方向。

**适用对象**

- 📊 **内容运营 / 舆情监测** — 快速掌握微博当前热门话题与舆论趋势
- 🎬 **创作者 / 博主** — 了解平台热点方向，找准内容选题和发布时机
- 🏢 **品牌 / PR** — 监控是否有品牌相关话题上榜

---

## 功能特性

### 核心功能

- **热搜榜单**：获取微博当前热搜榜单，一站式掌握全平台热点。
- **排名展示**：按热搜排名逐条展示，附带热度值。
- **一键跳转**：每条热搜附带 `https://s.weibo.com/weibo?q=` 搜索链接，点击直达。
- **灵感分析**：对热搜内容按赛道归类，提炼选题方向与创作建议，附带风险提示。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) 提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 查看热搜 | 「微博热搜」/ 「今天热搜有什么」 | 获取当前微博热搜榜单 |
| 查看指定热搜 | 「看看热搜第一是什么」 | 按排名查找特定热搜 |
| 订阅每日推送 | 「确认订阅」 | 创建定时任务，每天09:00自动推送 |

### 输出示例

> 🔥 微博当前热搜榜，共 **50 条**热搜：

| # | 热搜标题 | 热度 |
|---|----------|------|
| 1 | [某某事件引发热议](https://s.weibo.com/weibo?q=...) | 305.2w |
| 2 | [某明星回应争议](https://s.weibo.com/weibo?q=...) | 198.5w |
| 3 | [某某电影票房破亿](https://s.weibo.com/weibo?q=...) | 126.3w |

> 💡 **创作灵感分析**
>
> **🔥 热点赛道分布**
> - 🎬 娱乐明星 / ⚽ 体育赛事 / 🌪️ 天气灾害
>
> **📈 内容选题建议**
> - **天气科普**：结合台风动态做避险指南
> - **演唱会大盘点**：整理近期演唱会做攻略
>
> **⚠️ 风险提示**
> - 政策类热搜不做主观评论
> - 灾害类需核对信息来源

> 📩 是否订阅「**微博热搜**」的每日推送？订阅后每天自动推送最新热搜榜（默认 09:00，支持自定义推送时间）。
