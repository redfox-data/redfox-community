# 小红书内容生成

## 简介

xhs-copywriter 是一款专为小红书内容创作者打造的一站式笔记生成工具，把「找对标 → 拆规律 → 写正文」串成一条完整流水线。

通过简单的一句话触发，你可以：

- 📊 **自动抓取爆款样本**：按关键词从多维榜单中获取当下热门内容，数据真实、规律鲜活
- 🧠 **AI 深度拆解规律**：提炼标题结构、开头钩子、干货密度与互动话术等可复用爆款公式
- ✍️ **直出完整文案**：生成推荐标题 × 6 + 正文 + 标签 + 公式来源，可直接发布，无需二次加工

适用于个人博主、品牌运营、MCN 机构等有持续内容产出需求的用户。

---

## 功能特性

### 🎯 核心功能

- **📡 多维爆款数据采集**：覆盖新手友好爆款、当日点赞爆款、当日增长爆款、持续增长爆款四大榜单维度，样本更全面
- **🧩 爆款公式自动提炼**：从标题模式、开头结构、内容节奏、互动设计等多维度拆解规律，不暴露原始数据
- **✍️ 完整文案一键生成**：输出推荐标题（6个）、正文、标签列表、爆款公式说明，格式即发即用
- **🌐 热点资讯自动融入**：生成前联网补充最新行业动态与热点词汇，内容不过时

### ✨ 特色亮点

- **🎨 风格自适应**：支持用户提供个人写作风格样本，生成内容更贴近本人语气
- **⚡ 无账号无 Key**：无需注册小红书账号或配置 API，安装依赖即可使用
- **📋 来源透明**：输出结果附带参考爆款笔记链接与数据来源说明，规律可溯源

## 使用指南

### 基础使用（3 步完成）

#### 第 1 步：说出你的需求

直接用自然语言告诉我你想要什么，例如：

> 「帮我写一篇关于夏日防晒的小红书笔记」

#### 第 2 步：（可选）提供风格参考

如果你有自己常用的写作风格，可以贴几段文字，文案会更贴近你的语气。没有也完全没关系，直接跳过即可。

#### 第 3 步：获取完整文案

工具自动完成数据抓取 → 规律拆解 → 内容生成，最终交付：

- 📌 推荐标题（6 个，含数字型 / 痛点型 / 悬念型等多种风格）
- 📝 完整正文（含开头钩子、正文模块、互动收尾）
- 🏷️ 标签列表（10 个左右，覆盖核心词 + 长尾词）
- 🔍 爆款公式来源（参考笔记链接 + 互动数据）

### 常用命令速查

| 命令示例                             | 功能                               |
| ------------------------------------ | ---------------------------------- |
| `帮我写一篇关于 [话题] 的小红书笔记` | 生成完整笔记文案                   |
| `帮我找 [话题] 的爆款笔记`           | 查找并展示近期热门对标内容         |
| `分析 [话题/领域] 的爆款规律`        | 输出该领域标题、结构、话术规律总结 |
| `生成 [话题] 方向的文案`             | 同「写笔记」，支持模糊话题输入     |

## 注意事项

- **完整流程**：工具会自动走完「抓数据 → 拆规律 → 联网补热点 → 生成文案」全流程，无需手动干预
- **数据来源**：爆款数据来自实时榜单脚本抓取，非人工编造；内容规律基于真实数据提炼，非 AI 臆想
- **合规输出**：生成内容符合小红书社区规范，不含违规导流、虚假宣传等风险表述
- **原始数据不外露**：工具不会将原始数据表格直接甩给用户，只呈现提炼后的结论与可用文案
