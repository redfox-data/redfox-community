# 小红书最新热门笔记 / xiaohongshu-realtime-search

---

## 简介

输入关键词就能搜到小红书上的热门笔记，支持按排序方式、发布时间、笔记类型自由筛选，笔记标题、作者、点赞、评论、分享、收藏数一览无余，点一下链接就能跳转看详情。

**核心价值**

- **内容新鲜不滞后**：每次搜索都直接拉取平台当前的最新笔记数据，不会拿旧缓存糊弄你。
- **筛选随心所欲**：能自由组合排序方式（综合 / 最新 / 最多点赞 / 最多评论 / 最多收藏）、发布时间（不限 / 一天内 / 一周内 / 一月内 / 一年内）、笔记类型（不限 / 视频笔记 / 普通笔记）。
- **翻页不遗漏**：结果多的时候可以一页页翻着看，不遗漏任何一条内容。
- **发布时间一目了然**：每条笔记都展示发布日期，方便快速判断内容时效性。

**适用对象**

- 🔍 **内容创作者** — 找同赛道热门笔记，研究对标内容的选题方向和爆款规律。
- 📊 **运营 / 数据分析** — 快速摸清某个品类在小红书上有哪些热门内容，掌握流量趋势。
- 🏢 **品牌 / MCN** — 筛选潜在合作达人，按互动量评估笔记价值，辅助投放决策。
- 🛒 **电商选品** — 搜索细分赛道的带货笔记，找到匹配的推广内容方向。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入任意关键词搜索小红书热门笔记。
- **多维排序**：综合/最新/最多点赞/最多评论/最多收藏五种排序，按不同维度发现优质内容。
- **时间筛选**：不限/一天内/一周内/一月内/一年内，灵活控制搜索时间窗口。
- **类型筛选**：不限/视频笔记/普通笔记，按内容形式精准过滤。
- **翻页浏览**：每页展示多条结果，支持多页翻看。
- **订阅推送**：支持关键词每日推送，定时获取最新搜索结果。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 搜某个品类笔记 | 「搜一下A股」/ 「看看职场穿搭」 | 按最多点赞、一月内搜索并展示结果 |
| 指定排序方式 | 「按最新发布搜索减脂餐」 | 切换为最新发布排序后搜索 |
| 指定时间范围 | 「搜索近一周的AIGC内容」 | 将发布时间筛选设为一周内 |
| 指定笔记类型 | 「搜一下视频笔记」 | 仅展示视频类型笔记 |
| 组合筛选 | 「按最多评论、一周内、视频笔记搜古法健身」 | 同时指定排序、时间、类型后搜索 |
| 翻页查看更多 | 表格展示后回复「下一页」 | 自动递增页码翻页，结果连续不重复 |
| 订阅每日推送 | 「确认订阅」 | 创建定时任务，每天 09:00 自动推送 |
