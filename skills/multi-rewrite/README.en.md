# Multi-Platform Copy Rewrite / multi-rewrite

---

## Introduction

Turn one source draft into copy tailored for **WeChat Official Accounts, Channels, Douyin, Kuaishou, Bilibili, Xiaohongshu, and Zhihu**—without hand-adapting tone and structure for each channel.

**Core Value**

- **One draft, many platforms**: Pick one or more targets—or say “all”—and get styled versions in one pass
- **Platform-native tone**: Each version follows that platform’s title habits, voice, structure, and tags—not mere paraphrasing
- **List-style delivery**: One block per platform, clearly separated, ready to copy into each channel
- **Natural language only**: Name the platform(s) and paste your text; no commands to memorize

**Who It’s For**

- 📱 **Creators / MCNs** — Repurpose the same theme across seven major platforms quickly
- 🏢 **Brand / marketing teams** — Keep one core message, express it per platform
- ✍️ **Content ops** — Turn long articles, campaigns, or product copy into channel-ready posts

---

## Features

### Core Capabilities

- **Single-platform rewrite**: Specify e.g. Douyin or Xiaohongshu and get one full styled piece
- **Multi-platform batch**: Select several platforms (space- or comma-separated) and receive each version as its own item
- **All platforms at once**: Say “全部 / all / 所有” for all seven supported platforms
- **Platform auto-detection**: Parses platform names from your message; if none are found, you’ll be asked which to use
- **Xiaohongshu tags**: The Xiaohongshu version includes `#` topic tags at the end

---

## Usage Guide

State the target platform(s) and the copy you want rewritten.

### Quick Reference

| Intent              | Example phrase                                                         | Result                                      |
| ------------------- | ---------------------------------------------------------------------- | ------------------------------------------- |
| Single platform     | “Douyin: Great weather today, perfect for a walk”                      | One Douyin-style piece                      |
| Multiple platforms  | “Douyin Xiaohongshu Zhihu: This sunscreen is light and non-greasy”     | Three separate styled versions              |
| All platforms       | “All: Anniversary sale—50% off, 3 days only”                           | Seven versions in a list                    |
| Platform not stated | “Rewrite this for every platform: [your draft]”                        | You’ll be asked which platforms to include  |

### Output Example

After rewriting, you receive a per-platform list, roughly like:

```text
## Rewrite Results

---

### 1. 📌 WeChat Official Account

[WeChat-style copy]

---

### 2. 📌 Douyin

[Douyin-style copy]

---

### 3. 📌 Xiaohongshu

[Xiaohongshu-style copy]

#tag1 #tag2 #tag3

---
```
