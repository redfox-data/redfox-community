# TikTok视频下载 / tiktok-video-downloader

---

## 简介

贴上 TikTok 视频链接就能拿到无水印下载直链，复制到浏览器或下载工具就能保存。支持单个或批量链接解析。www.tiktok.com 网页链接和 vm.tiktok.com 短链都能认，不会让你折腾半天还下载到带水印的版本。

**核心价值**

- **贴链即得不费劲**：贴上视频链接就能拿到视频下载直链，复制到浏览器或下载工具就能保存，不用装插件、不用开会员。
- **批量解析更高效**：一次粘贴多个 TikTok 视频链接（空格分隔），批量解析并在最后汇总成功与失败数量，不用一条条重复操作。
- **链接智能校验**：自动判断输入的链接是否为 TikTok 视频链接，非 TikTok 链接会提示「请输入正确的 TikTok 视频链接」并终止流程，需要重新输入正确的链接。
- **视频干净无水印**：返回的都是无水印视频，收藏、二次创作都能直接拿去用。
- **两种链接都能认**：www.tiktok.com 网页链接和 vm.tiktok.com 短链都能识别，手机分享链接、电脑网页链接贴过来都能解析。
- **结果一眼看清**：资源类型、时长、下载链接、封面图、音频链接、内容描述分开展示，拿到手就知道该复制哪一个。

**适用对象**

- 🎬 **创作者 / 剪辑师** — 快速保存 TikTok 上的视频素材，用于二次创作
- 📚 **内容收藏者** — 把喜欢的 TikTok 视频备份到本地，不怕原视频被删
- 🔍 **运营 / 研究者** — 下载竞品或热点视频，慢慢分析

## 功能特性

### 核心功能

- **视频解析**：粘贴 TikTok 视频链接，一键解析出无水印视频下载链接。
- **批量解析**：支持一次粘贴多个 TikTok 视频链接（空格分隔），逐个解析并在最后汇总成功与失败数量。
- **链接校验**：自动识别输入的链接是否为 TikTok 视频链接，非 TikTok 链接会提示并终止流程，需要重新输入正确的链接。
- **信息完整**：展示资源类型、时长、下载链接、封面图、音频链接，内容描述完整逐行展示不截断。
- **双链接支持**：www.tiktok.com 网页链接和 vm.tiktok.com 短链都能识别。
- **链接时效提醒**：解析成功后返回的下载链接有效期约 5 分钟，会提醒用户立即复制使用。

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`) 提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。用户提供正确的 TikTok 视频链接时直接执行解析，无需额外确认。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 下载 TikTok 视频 | 「下载这个视频 https://www.tiktok.com/@user/video/xxxxx」 | 解析链接并返回视频下载直链 |
| 批量下载 TikTok 视频 | 「帮我下载这几个视频 链接1 链接2 链接3」 | 批量解析多个链接，逐个返回下载直链并汇总结果 |
| 保存 TikTok 视频 | 「帮我把这条 TikTok 视频存下来」 | 引导你贴上链接，解析后返回下载链接 |

### 输出示例

单个链接解析完成后，你将收到如下格式的结果：

> ✅ 解析成功！
>
> 📝 内容描述：视频文案完整展示
>
> 🎬 资源列表：
> 类型、时长、下载链接、封面图、音频链接一一列出
>
> 复制链接到浏览器或下载工具即可下载。
>
> ⚠️ 视频下载链接有效期约 5 分钟，请立即复制到浏览器打开或下载！

批量解析时，每个链接的结果会依次展示，最后还会汇总成功与失败数量：

> 📋 批量解析完成
> ✓ 成功：3 个
> ✗ 失败：1 个
>
> 复制链接到浏览器或下载工具即可下载。

如果输入了非 TikTok 的链接，会看到提示：

> [✗] 该链接不是 TikTok 视频链接，请输入正确的 TikTok 视频链接
> 正确的链接格式示例：
> https://www.tiktok.com/@user/video/xxxxx
> https://vm.tiktok.com/xxxxx/
