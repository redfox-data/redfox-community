# 小红书作品分析助手 / xhs-note-analyzer

---

## 简介

一键查询任意小红书笔记的完整详情，并以专业创作者视角进行多维度深度分析，帮助你快速了解作品全貌、获取可操作的优化建议。

**核心价值**

- **一站式查询**：只需提供笔记ID或链接，即可获取标题、正文、互动数据、作者信息等完整详情，无需登录或手动复制。
- **多维度深度分析**：从爆款成因、标题SEO、标签质量、互动解读、封面视觉、转化潜力等维度进行专业分析，不止看数据，更看懂数据背后的逻辑。
- **批量高效**：支持同时查询多篇笔记，满足竞品对比、赛道研究等批量分析需求。

**适用对象**

- 📝 **小红书博主** — 复盘自身作品表现，获取针对性优化建议。
- 🛍️ **品牌 / MCN 运营** — 分析竞品笔记策略，辅助合作决策。
- 📊 **内容策划** — 批量拆解爆款规律，沉淀可复用的方法论。

---

## 功能特性

### 核心功能

- **作品详情查询**：获取标题、正文、封面图、作品类型、互动数据（点赞/收藏/评论/分享）、作者信息、话题标签等完整信息。
- **多维度作品分析**：以专业创作者视角，从主题匹配度、爆款成因、信息结构、标题SEO、标签质量、互动解读、封面视觉、转化潜力等维度深度拆解。
- **封面图视觉分析**：对封面图进行视觉冲击力、构图色调、文字叠加效果等多维度分析。
- **账号定位判断**：结合笔记内容与账号信息，推断账号定位方向，评估本篇内容与账号调性的匹配度。
- **批量查询**：支持同时提供多个笔记ID或链接，一次查询多篇笔记，单个失败不影响其余。
- **数据可信度提示**：当互动数据为平台估算值时，自动醒目提示，避免误导决策。

## 使用指南

直接用自然语言描述需求即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 分析单篇笔记 | "帮我分析一下这个小红书作品" | 查询完整详情并进行多维度深度分析 |
| 按ID查询 | "查询小红书作品 6a23eddb000000003502a39d" | 直接按ID获取笔记数据与分析 |
| 按链接查询 | 粘贴笔记分享链接 | 自动提取ID并查询 |
| 批量查询 | 同时提供多个笔记ID或链接（逗号分隔） | 一次查询多篇，逐一返回结果 |

### 输入方式

- **直接给ID**：`6a23eddb000000003502a39d`
- **给链接**：支持标准笔记链接和 discovery 链接，自动提取ID
- **批量输入**：多个ID或链接用逗号分隔即可

> 注：短链接（xhslink.com）暂不支持，请提供完整链接。
