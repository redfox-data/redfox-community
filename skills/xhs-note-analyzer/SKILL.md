# 小红书作品分析助手

快速查询任意小红书笔记的完整信息，并以专业创作者视角进行多维度深度分析，帮助你高效了解单篇作品全貌并获取可操作的优化建议。

## 功能特性

| 功能 | 说明 |
|------|------|
| 作品基础信息 | 标题、正文描述、封面图、作品类型（视频/图文）、IP属地、最后编辑时间 |
| 互动数据 | 点赞、收藏、评论、分享、总互动数 |
| 作者信息 | 作者昵称、作者ID、作者头像 |
| 媒体资源 | 笔记链接、图片列表、视频地址（视频类型）、视频时长 |
| 话题标签 | 笔记关联的全部话题标签与标签列表 |
| 跳转链接 | 作者主页、笔记原文、封面图均可点击跳转 |
| 作品内容分析 | 从主题匹配度、爆款成因、信息结构、标题SEO、标签质量、互动解读、封面视觉、转化潜力等维度深度分析，含账号定位判断与优化建议 |
| 封面图视觉分析 | 对封面图进行视觉冲击力、构图色调、文字叠加等多维度分析 |
| 数据可信度提示 | 当互动数据为估算值时，醒目提示数据偏差风险 |

## 使用指南

### 1. 鉴权配置

前往 [红狐hub](https://redfox.hk/settings/api-keys?source=github) 获取 API KEY，配置设备环境变量 `REDFOX_API_KEY` 后即可使用。

### 2. 触发方式

使用以下任意说法即可触发本技能：

- "帮我分析一下这个小红书作品"
- "查询小红书作品 6a23eddb000000003502a39d"
- "小红书作品分析"
- "获取小红书笔记数据"
- "xhs作品分析"

### 3. 输入方式

支持多种输入方式，自动识别并提取笔记ID：

- **直接提供笔记ID**：如 `6a23eddb000000003502a39d`
- **标准笔记链接**：如 `https://www.xiaohongshu.com/explore/6a23eddb000000003502a39d`
- **discovery链接**：如 `https://www.xiaohongshu.com/discovery/item/6a23eddb000000003502a39d`
- **批量查询**：同时提供多个笔记ID或链接（逗号分隔），一次查询多篇笔记

> 注：短链接（xhslink.com）暂不支持，请提供完整链接。

## 常见问答

**Q：支持查询哪些类型的笔记？**
A：支持图文笔记和视频笔记两种类型，视频笔记会额外展示视频地址和时长。

**Q：数据是实时的吗？**
A：是的，每次查询都会获取最新数据。

**Q：查询失败怎么办？**
A：请先确认笔记ID是否正确、笔记是否仍可公开访问，以及API KEY是否已正确配置。

**Q：内容分析是基于什么进行的？**
A：分析基于查询到的真实数据字段（正文、标签、互动数等），不做无依据的推测；缺失字段会自动跳过对应分析。

## 数据范围说明

- 本技能仅支持查询**公开可见**的小红书笔记
- 已删除、设为私密、或被平台限流的笔记可能无法获取完整数据
- 互动数据为查询时刻的**实时快照**，不反映历史变化趋势
