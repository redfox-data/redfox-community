# 红狐Skill生成器 / redfox-skill-generator

---

## 简介

帮你从零开始快速创建符合红狐规范的完整 Skill 文件包。只需用自然语言描述功能需求、提供接口地址与接口文档，即可一键生成结构规范、内容专业的全套技能文件，省去反复查阅规范和手动排版的时间。

**核心价值**

- **说需求即生成**：用日常语言描述你想要的功能，无需了解任何规范细节或技术名词，生成器自动理解并输出可用的 Skill
- **接口文档即用**：直接把接口文档丢进来，自动解析参数与调用方式，生成即用的对接说明
- **一次到位**：一次性输出全套 Skill 文件，自动完成质量校验，拿到就能用

**适用对象**

- 👨‍💻 **开发者** — 快速将接口能力封装为红狐 Skill，省去查阅规范与手动排版的时间
- 📊 **运营 / 产品** — 无需懂代码也能创建专属 Skill，描述需求即可
- 🏢 **技术负责人** — 统一团队 Skill 质量标准，确保所有输出结构规范一致

## 功能特性

### 核心功能

- **需求智能分析**：用日常语言描述即可，自动识别 Skill 核心能力
- **接口文档驱动**：提供接口地址和文档，自动解析参数与调用规范
- **智能方案匹配**：根据需求自动选择最合适的文件组织方式
- **全方位资源组织**：自动分析并合理规划所需的配套脚本与文档
- **开箱即用交付**：一次性生成全套可用 Skill 文件，无需额外修整
