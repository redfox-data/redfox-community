# 抖音涨粉账号推荐 / douyin-rise-ranking

---

## 简介

专注抖音涨粉数据查询，实时追踪各领域账号的粉丝增长排行，支持日榜/周榜/月榜多时间维度与 27 个垂直分类精准筛选，自动生成涨粉趋势分析与热点影响力量化报告。

**核心价值**

- 多维时间榜单：日榜/周榜/月榜三种粒度，覆盖日常监测与长周期复盘
- 27 类垂直筛选：从美食到游戏，按赛道精准锁定涨粉排行
- 智能趋势分析：自动分析 TOP50 涨粉数据，量化热点事件的涨粉拉动效果
- 定时订阅推送：一键订阅，数据按更新时间自动送达

**适用对象**

- 📱 内容创作者 — 了解赛道涨粉热点，获取选题灵感
- 🏢 品牌方 — 追踪同赛道竞品账号涨粉表现，掌握营销节奏
- 📊 MCN 机构 — 批量监测达人涨粉数据，发现潜力账号
- 🎯 广告投放/商务 — 筛选高增长潜力账号，提升选号效率

## 功能特性

### 核心功能

- 多维时间榜单：日榜/周榜/月榜三种维度，覆盖不同运营周期需求
- 27 类垂直分类筛选：美食、科技、美妆等，按领域精准筛选涨粉排行
- 智能趋势分析：分析 TOP50 涨粉趋势，量化热点事件对涨粉的拉动效果
- 定时订阅推送：一键订阅日榜/周榜/月榜，更新时间自动送达
- 账号可点击跳转：榜单含账号主页直达链接，方便一键查看

### 特色亮点

- 实时更新：日榜每日 18:00、周榜每周一 18:00、月榜每月 3 号 18:00 准时更新
- 可视化呈现：TOP3 金银铜牌标识，涨粉率与粉丝增量一目了然
- 深度洞察：涨粉趋势分析 + 热点影响力量化，两段式分析覆盖全量 TOP50
- 安全便捷：仅需 API Key，无需账号密码，数据通过标准化接口获取

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| --- | --- | --- |
| 查询涨粉日榜 | 查一下美食类的抖音涨粉榜 | 输出美食分类日榜 TOP20 + 涨粉趋势分析 + 订阅提示 |
| 查询本周涨粉 | 这周科技类涨粉排行 | 自动使用周榜，匹配数码科技分类 |
| 查询本月涨粉 | 本月美妆类涨粉榜 | 自动使用月榜，匹配化妆美容分类 |
| 按关键词查询 | 宠物类涨粉榜 | 自动匹配动物分类，支持模糊关键词 |
| 查看更多排名 | 查看更多 | 展示 TOP21-50 的剩余 30 条数据 |
| 订阅定时推送 | 订阅日榜 / 订阅周榜 / 订阅月榜 | 按时自动推送最新涨粉数据 |

### 订阅方式

查询完成后会提示订阅选项，回复数字 1-5 即可：

1️⃣ 订阅日榜 — 每日 18:00 推送
2️⃣ 订阅周榜 — 每周一 18:00 推送
3️⃣ 订阅月榜 — 每月 3 号 18:00 推送
4️⃣ 全部订阅 — 日/周/月均推送
5️⃣ 暂不订阅

也支持直接说订阅日榜 / 订阅周榜 / 订阅月榜 / 全部订阅。

## 重要数据说明

| 榜单类型 | 可回溯范围 | 更新时间 |
| --- | --- | --- |
| 日榜 | 近 7 天 | 每日 18:00 更新前一日数据 |
