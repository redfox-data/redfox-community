# 抖音涨粉榜 Skill

> 统计抖音平台账号日/周/月三个时间维度的粉丝增量，T+1 数据更新，生成标准化涨粉排行榜。

## 功能简介

| 时间维度 | 说明 |
|----------|------|
| 日榜 | 统计昨日粉丝增量（T+1） |
| 周榜 | 统计近7天粉丝增量（T+1） |
| 月榜 | 统计近30天粉丝增量（T+1） |

**输出字段**：抖音号 · 总粉丝数 · 粉丝增量 · 涨粉率

## 数据说明

- **数据源**：红狐 API（唯一可信数据源）
- **更新频率**：T+1，当日数据最早次日可查
- **认证**：使用环境变量 `COZE_REDF0X_API7641879691305091122` 作为 `X-API-KEY`
