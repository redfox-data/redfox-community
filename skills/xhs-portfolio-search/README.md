# 小红书账号作品追踪 / xhs-portfolio-search

---

## 简介

小红书账号作品列表实时查询工具，输入博主主页链接即可查看最新发布的笔记作品，**数据实时获取**，确保你看到的每一条都是博主最新发布的内容。支持翻页浏览更多历史作品，还可设置每日/每周/自定义时间自动推送，轻松掌握博主的完整内容动态。

**核心价值**

- **实时数据**：每次查询均从平台实时获取最新作品，非缓存、非历史快照，确保数据时效性。
- **专属平台**：专注于小红书单一平台，数据精准、响应快速，不做跨平台泛化。
- **持续追踪**：支持定时订阅推送，无需反复手动查询，博主更新自动送达。

**适用对象**

- 📊 **品牌 / MCN 运营** — 实时追踪合作博主或竞品账号的作品发布动态。
- 📝 **内容创作者** — 关注同赛道博主的完整内容脉络，获取创作灵感。
- 🔍 **数据分析师** — 获取博主全量作品数据，用于内容表现评估与趋势分析。

---

## 功能特性

### 核心功能

- **实时作品查询**：输入博主主页链接或 userId，即时获取最新笔记列表，数据为平台实时返回，非缓存数据。
- **翻页浏览**：逐页查看博主更多历史作品，深入了解完整内容脉络。
- **多维数据**：返回点赞、收藏、评论、分享等完整互动数据，全面评估每条作品表现。
- **定时订阅推送**：支持每日、每周或自定义时间自动查询并推送最新作品，持续追踪博主更新。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求即可，无需记忆固定命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| --- | --- | --- |
| 查询作品列表 | 「查一下小红书用户 5a73c5fa4eacab4c4ccc9778 的作品列表」 | 实时获取该博主最新笔记作品及互动数据 |
| 通过链接查询 | 「帮我看看这个账号的笔记：xiaohongshu.com/user/profile/5a73c5fa4eacab4c4ccc9778」 | 自动识别链接并返回作品列表 |
| 翻页查看更多 | 「翻页，看下一页作品」 | 展示博主更多历史作品 |
| 每日订阅 | 「订阅这个博主，每日推送」 | 每天自动查询并推送最新作品 |
| 每周订阅 | 「每周推送一次」 | 每周自动查询并推送最新作品 |
| 自定义时间 | 「每周三 14:00 推送」 | 按指定时间自动推送 |
