# 小红书账号作品追踪

## 简介

小红书账号作品列表实时查询工具，根据小红书用户的主页链接获取该用户最新发布的笔记作品列表，返回实时数据。支持翻页浏览更多作品，还支持每日/每周/自定义时间订阅自动推送，轻松掌握博主的完整内容动态。

## 功能特性

| 功能模块 | 能力描述 | 核心价值 |
|---------|---------|---------|
| 实时查询 | 输入 主页链接/ userId 实时获取博主作品列表 | 获取非缓存的实时数据 |
| 翻页浏览 | 逐页查看博主更多历史作品 | 深入了解博主完整内容 |
| 多维数据 | 返回点赞、收藏、评论、分享等完整数据 | 全面评估作品表现 |
| 定时订阅 | 支持每日/每周/自定义时间自动推送 | 持续追踪博主更新动态 |

## 一键安装

### 前置条件

- 已获取红狐 hub 的 API Key（`REDFOX_API_KEY`）

### 获取 API Key

请前往 [红狐hub](https://redfox.hk/settings/api-keys?source=github) 获取 API KEY

### 配置 API Key

方案1: 以 Qoder 为例，将 REDFOX_API_KEY 添加到 `~/.openclaw/openclaw.json` 中：

```bash
{ "env": { "REDFOX_API_KEY": "ak_xxxx..." } }
```

方案2: 终端配置

```bash
export REDFOX_API_KEY="ak_xxxx..."
```

## 使用指南

直接用自然语言描述需求即可，例如：

- 「查一下小红书用户 5a73c5fa4eacab4c4ccc9778 的作品列表」
- 「帮我看看这个小红书账号的笔记：xiaohongshu.com/user/profile/5a73c5fa4eacab4c4ccc9778」
- 「翻页，看下一页作品」
- 「订阅这个博主，每日推送」
- 「每周推送一次」
- 「每周三 14:00 推送」
