# 多平台违禁词检测 / multi-wordcheck

---

## 简介

基于官方违禁词库，覆盖公众号、小红书、抖音三大内容平台的审核标准，支持文案、文件、图片、链接多种输入形式，快速输出违禁词标记与结合上下文的替换建议。

**核心价值**

- 三大平台独立词库，切换平台一句话搞定，发布前精准排雷
- 不止标记违禁词，还给出结合语境的替换词与修改理由，直接可用
- 自动生成替换后的可发布版本，复制即发，省去手动逐词修改的工作量

**适用对象**

- ✍️ 自媒体创作者 — 发布前自查，降低被平台限流或驳回的风险
- 📢 品牌运营 / 电商 — 批量排查活动文案、商品详情页，统一合规标准
- 🏢 MCN 机构 / 内容审核团队 — 多人协作场景下标准化合规流程

---

## 功能特性

### 核心功能

- **违禁词标记**：原文中命中词以加粗标出，逐一列出违禁词数量与类型
- **上下文替换建议**：每个违禁词给出结合语境的替换词与更换理由，而非机械关键词替换
- **可发布版本输出**：自动生成替换后的完整文案，替换处以加粗斜体标出，语义通顺可直接复制
- **多平台切换**：公众号（默认）、小红书、抖音三套独立词库，输入时附带平台关键词即可切换
- **多形式输入**：支持直接粘贴文案、上传 txt 文本文件、上传图片（自动提取图中文字）、粘贴网页链接（自动抓取正文）
- **长文案分批检测**：超过 3000 字自动提示分批选项，支持逐批检测后合并结果

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图           | 示例话术                                           | 效果                                                          |
| -------------- | -------------------------------------------------- | ------------------------------------------------------------- |
| 检测公众号文案 | `帮我查一下这段公众号文案有没有违禁词：[文案内容]` | 按公众号规则检测，输出标记结果与替换建议                      |
| 切换小红书平台 | `小红书这个文案帮我查一下：[文案内容]`             | 切换为小红书词库检测                                          |
| 切换抖音平台   | `抖音脚本违禁词检测：[脚本内容]`                   | 切换为抖音词库检测                                            |
| 上传文件检测   | 直接上传 txt 文件或图片                            | 自动读取内容并执行检测，无需额外操作                          |
| 网页链接检测   | `检测这个页面的文案：https://...`                  | 自动抓取页面正文后检测                                        |
| 长文案分批     | 超过 3000 字时系统自动提示                         | 回复 `1` 取前 3000 字 / 回复 `2` 自动分批合并 / 回复 `3` 取消 |

### 输出示例

检测到违禁词时，按三个板块输出：

1. **违禁词检测结果** — 原文（违禁词加粗标出）+ 违禁词数量与类型
2. **修改建议** — 违禁词 / 替换词 / 更换理由对照表
3. **建议优化文案** — 替换后完整可发布版本（替换处加粗斜体）

未检测到违禁词时，仅输出"未检测到违禁词，文案内容合规"。

## 使用场景

| 场景             | 角色         | 示例问法                            | 收益                                                 |
| ---------------- | ------------ | ----------------------------------- | ---------------------------------------------------- |
| 推文发布前自查   | 公众号运营   | `群发前帮我检测一下这篇推文`        | 排查广告法禁用词与虚假宣传表述，降低被驳回或限流风险 |

| 短视频脚本筛查   | 抖音编导     | `抖音口播脚本检测：[脚本]`          | 筛查敏感词与违规表述，确保成片通过平台审核           |
| 营销物料批量检查 | 品牌电商运营 | `检测这个落地页的文案：https://...` | 快速完成多页面合规筛查，输出统一格式的检测报告       |
