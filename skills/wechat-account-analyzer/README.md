# 公众号账号诊断 / wechat-account-analyzer

---

## 简介

对任意公众号账号进行四维度量化评分，对标行业平均水平，输出可落地的运营优化建议，让公众号运营告别主观判断、用数据说话。

**核心价值**

- **四维度量化诊断**：内容健康度、用户活跃度、核心数据表现、运营规范性，每项百分制评分，账号全貌一次看清
- **行业对标**：以红狐指数为锚点，自动匹配同赛道对标账号，横向找差距
- **分级优化建议**：紧急 / 重点 / 持续三级优先级，每条建议含预期量化效果，直接可执行
- **多账号对比**：支持同时诊断多个账号，生成横向对比报告

**适用对象**

- 📱 **公众号号主 / 内容创作者** — 一次诊断明确账号短板，量化改进方向
- 📊 **新媒体运营 / 品牌方** — 数据驱动内容策略，摆脱经验式运营
- 🏢 **MCN 机构** — 批量评估旗下或潜在签约账号商业价值

---

## 功能特性

### 核心功能

- **四维度评分**：内容健康度 + 用户活跃度 + 核心数据 + 运营规范性，综合生成百分制总分及 S/A/B/C/D/E 六级评级
- **近期作品数据汇总**：近7天作品阅读、点赞、评论、在看数据表格，一目了然
- **分级优化建议**：按紧急 / 重点 / 持续三个优先级输出可执行建议，每条含预期量化效果
- **同赛道对标账号**：自动匹配相似账号，横向对比找差距与成长路径
- **多账号对比诊断**：支持同时输入多个账号，生成并排对比报告

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言说出账号名称或诊断需求即可，无需记忆命令格式。

### 常用说法速查

| 意图           | 示例话术                           | 效果                                        |
| -------------- | ---------------------------------- | ------------------------------------------- |
| 单账号诊断     | 「诊断"十点读书"公众号」           | 输出完整五章节诊断报告，含评分、建议与对标  |
| 多账号对比     | 「对比诊断"十点读书"和"洞见"」     | 分别诊断并生成横向对比总结表格 + 差异化建议 |
| 通过账号ID查询 | 「诊断公众号ID"gh_xxxxxxxx"」      | 直接以公众号ID精准查询，避免同名混淆        |
| 账号自检       | 「帮我看看这个公众号数据表现如何」 | 输出诊断报告，标注行业对标水平和优化优先级  |

### 输出示例

完成诊断后将收到5章节固定格式报告：

**一、账号信息** — 账号名称（可跳转）、简介、粉丝量等基础信息

**二、综合评分** — 四维度百分制评分 + 综合评级（S/A/B/C/D/E）+ 行业对标

**三、近7天作品数据** — 作品标题（可跳转）、阅读 / 点赞 / 评论 / 在看数据表格

**四、优化建议** — 紧急 / 重点 / 持续三级建议，每条含预期量化效果

**五、行业对标分析** — 自动匹配同赛道相似账号，横向对比找差距

| 场景         | 角色         | 示例问法                             | 收益                                       |
