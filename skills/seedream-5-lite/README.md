# seedream5.0 lite / seedream-5-lite

---

## 简介

基于火山方舟 **seedream 5.0 lite** 模型的 AI 图片生成工具，输入一句话即可生成高质量图片，支持文生图、图生图、组图生成与提示词优化。内置免费额度，开箱即用。

**核心价值**

- **零门槛上手**：内置公共 API Key，约 **10000 次免费额度**，无需注册、绑卡或配置密钥。
- **一行命令出图**：粘贴提示词即可生成图片，无需学习复杂参数。
- **多模式覆盖**：文生图、图生图、组图生成，满足从单张创意到批量素材的全场景需求。

**适用对象**

- 🎨 **设计师 / 插画师** — 将创意想法快速可视化为高质量图片。
- 📱 **社交媒体创作者** — 批量生成配图、封面图、产品展示图。
- 🛒 **电商运营** — 零摄影成本产出统一风格的商品场景图。
- 💡 **AI 爱好者** — 体验最新 seedream 5.0 lite 模型，探索 AI 绘图能力。

---

## 功能特性

### 核心功能

- **文生图**：输入中文或英文描述，自动生成高清图片。
- **图生图**：上传参考图 + 提示词，基于原图进行编辑和风格转换。
- **组图生成**：支持自动模式一次生成多张关联图片（最多 15 张）。
- **提示词优化**：内置标准（高质量）和快速两种优化模式，提升出图效果。
- **高分辨率输出**：支持 2K / 3K / 4K 超高清分辨率，默认 2048×2048。
- **任务管理**：支持仅提交任务获取 taskId，随时查询进度和结果。
- **自动轮询下载**：自动等待任务完成并下载图片，带进度条显示。

- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。
