# YouTube 提文案 / youtube-digest

---

## 简介

输入 YouTube 视频链接或 ID，一键提取视频文字稿，非中文字幕自动翻译为中文，终端表格展示 + Markdown 存档 + 可导出 Excel。

**核心价值**

- **一键提文案**：丢一个链接就能拿到完整口播文案，省去手动抄录。
- **自动翻译**：英文或其他语言字幕自动翻译为中文，无需额外操作就能看懂。
- **多格式输出**：终端表格速览、Markdown 存档归档、Excel 导出分享，三种交付方式随你选。

**适用对象**

- 📝 **内容创作者** — 提取同行视频文案，分析话术结构与选题方向。
- 🎓 **学习者 / 研究者** — 把教程、演讲、播客转为可搜索的文字笔记。
- 📊 **运营 / 市场团队** — 批量提取竞品视频内容，快速完成内容调研与二次创作。

---

## 功能特性

### 核心功能

- **三种输入格式**：完整 URL、`youtu.be` 短链、纯视频 ID，自动识别，丢哪个都能用。
- **中文轨优先**：优先选中文字幕轨，有中文字幕直接输出；没有中文轨时回退英文并自动翻译为中文。
- **自动翻译**：非中文字幕自动翻译为中文（基于 Google Translate，批量合并提升效率）；加 `--no-translate` 可保留原文。
- **默认纯文案**：不带时间戳，拿到就是干净可读的全文；`--timestamp` 随时切换带时间戳版本。
- **Excel 导出**：`--excel` 一键导出 xlsx（标题/时长/视频链接/文案内容），自动换行、列宽适配。
- **表格化交付**：标题/时长/文案内容三字段表格呈现，信息一目了然。
- **稳健重试**：网络异常、数据为空自动递增延迟重试，限频自动等待后重试。

- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。

## 使用指南

直接用自然语言说出需求即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 提取视频文案 | "帮我提一下这个视频的文案" + 链接 | 自动提取并翻译为中文，表格展示 |
| 带时间戳提取 | "给我带时间戳的文案版本" | 输出 `[MM:SS]` 时间戳 + 文案，方便定位原片 |
| 导出 Excel | "帮我导出 Excel" | 生成 xlsx 文件（标题/时长/链接/文案） |
| 保留原文不翻译 | "保留英文原文，不要翻译" | 加 `--no-translate`，输出字幕原文 |
| 指定语言 | "只要英文字幕" | 按指定语言优先级选轨 |

### 输出示例

提取完成后，你将看到类似以下格式的表格：

| 标题 | 时长 | 文案内容 |
|------|------|----------|
| Rick Astley - Never Gonna Give You Up (Official Video) (4K Remaster) | 03:31 | 我们对爱情并不陌生 / 你知道规则，我也一样 / ……（中文译文全文） |
