# A股情报调查员 / ai-intelligence-investigator

---

## 简介

深度A股情报调查与分析工具，帮你快速完成公司研究、竞品对比、事件追踪、背景核实和信息验证，输出结构化调查报告，为投资决策、竞品分析、舆情追踪、背景调查提供可靠的信息支撑。

**核心价值**

- **全面覆盖**：A股公司基本面、行业产业链、概念热点、资金流向、公告研报、舆情动态一网打尽
- **可信度标注**：调查结果附带可信度等级，区分已确认、待确认、已否定，帮你判断信息可靠程度
- **结构化交付**：输出完整调查报告，含数据、来源、可信度标注，可直接用于决策参考
- **多场景适用**：A股情报、竞品分析、事件追踪、背景调查、信息验证，一个工具覆盖多种调查需求

**适用对象**

- 📈 **投资者** — 快速获取A股公司基本面、行业格局与资金动态全景分析
- 🏢 **产品经理/市场人员** — 系统了解竞品功能、用户口碑与市场表现
- 📰 **内容运营/媒体人** — 追踪热点事件时间线，掌握舆论走向与多视角信息
- 🤝 **企业BD/投资人** — 商务合作前背景调查与信誉评估
- 🔍 **所有人** — 验证网络信息真伪，确认关键事实

## 功能特性

### 核心功能

- **A股情报调查**：获取上市公司财报、公告、研报、行业政策、资金流向、股吧舆情等全面信息，输出涵盖公司概况、财务表现、行业赛道、概念热度、资金动态、机构观点、风险提示的结构化报告
- **竞品情报调查**：从产品功能、用户口碑、市场表现多维度分析竞品，覆盖国内外主流平台
- **舆情事件调查**：还原事件经过，收集多方观点，重建事件时间线，识别关键信息可信度
- **人物背景调查**：核实基本信息，验证专业背景，排查争议与信誉风险
- **信息交叉验证**：多方比对同一信息，给出确认/待确认/否定的结论与偏差分析
- **可信度标注**：为调查结果标注可信度等级（✅已确认/⚠️待确认/❌已否定/🔍单一来源），帮你快速判断信息可靠程度

### 特色亮点

- **全方位情报覆盖**：覆盖上市公司基本面、行业产业链、概念热点、资金流向、研报评级等核心维度
- **可信度透明**：每条关键信息标注来源与可信度，让你清楚知道哪些信息可靠、哪些需进一步验证
- **A股深度聚焦**：专为A股市场设计，深度覆盖东方财富、雪球、巨潮资讯等专业财经信息源

## 使用指南

直接用自然语言描述你的调查需求即可，无需记忆固定命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 调查A股公司 | 「帮我调查一下宁德时代这家公司」 | 输出公司基本面、财务、行业地位全景分析报告 |
| 调查竞品 | 「帮我调查一下 Notion 这个产品」 | 输出竞品功能、用户口碑与市场表现分析报告 |
| 追踪热点事件 | 「追踪最近的XX事件舆论走向」 | 还原事件时间线，收集多方观点与舆论分布 |
| 背景调查 | 「调查一下XX公司创始人的背景」 | 核实基本信息，排查争议与信誉风险 |
| 验证信息 | 「验证"XX公司获得10亿融资"是否属实」 | 多方核实，给出确认/否定/待确认结论 |
