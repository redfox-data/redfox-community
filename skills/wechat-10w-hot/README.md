## 简介

精准锚定 **10w+** 爆款文章。爆文捕手为您深度拆解公众号全网爆款模型，不仅提供每日 **10w+** 榜单，更通过数据洞察揭示刷屏背后的创作逻辑。

**功能介绍**

**全网爆文一键捕获**：彻底解决爆文搜索难、效率低的痛点，每日实时收录全网各行业 **10w+** 阅读量 TOP 50 热门文章，瞬间掌握头部流量走向。

**垂直领域精准锁定**：深度细分行业赛道，支持搜索与订阅特定领域的 **10w+** 资讯，帮你实时盯紧竞品动向，确保选题永不偏航。

**爆款基因深度解码**：自动拆解爆文的吸睛标题、内容叙事、发布时段等核心维度，将感性的内容转化为理性的创作模型，助你破解流量密码。

**沉浸式阅览与导出**：接近公众号原生的网页阅读体验，支持一键导出为 PDF 文档，方便团队离线复盘与深度拆解学习。

**核心价值**

- **榜单即所得**：按分类与时间窗口输出完整爆款榜单，对话内即可阅读，并生成可分享的展示页。
- **数据可核对**：基于每日更新的热门数据，少于 10 条如实展示，不凑数、不虚构。
- **规律可复盘**：在榜单后给出基于真实条目的爆款规律分析，并支持按赛道订阅推送（产品侧若已开通）。

**适用对象**

- 📝 **公众号编辑 / 运营** — 追热点、看标题与账号分布，快速锁定 10w+ 供给。
- 📊 **内容负责人** — 需要可分享、可打印的榜单页，便于团队复盘。
- 📈 **增长 / 商务** — 按赛道盯竞品与行业 10w+ 爆文动向。

## 功能特性

### 核心功能

- **热门推荐**：每日更新公众号 **10w+** 阅读文章榜单，支持总榜与多分类查询。
- **可视化展示**：将榜单内容美化后呈现，支持导出分享或保存为 PDF。
- **订阅推送**：在平台支持的前提下，可按赛道订阅并按时接收最新热门文章。

### 特色亮点

- **总榜 + 分赛道**：既可看全品类热门，也可锁定科技、财经、健康等垂直领域。
- **爆款规律分析**：不只给列表，还帮你从真实爆文中提炼可复用的创作规律。

## 使用指南

直接用自然语言说出你的需求即可，无需记忆固定命令。

### 常用说法速查

| 意图         | 示例话术                             | 效果                                     |
| ------------ | ------------------------------------ | ---------------------------------------- |
| 看今日爆文   | 「今天的 10w+ 爆文有哪些？」         | 输出最新可用日期的总榜榜单               |
| 查垂直赛道   | 「科技类最近一周有哪些 10w+ 热门？」 | 按分类与时间窗口输出对应榜单             |
| 多看几条     | 「展开全部，给我看 50 条」           | 在同时间范围内展示更多条目               |
| 要可分享页面 | 「生成可以导出 PDF 的榜单页」        | 在对话榜单基础上生成可视化页面，支持导出 |
| 订阅赛道     | 「帮我订阅财经赛道的每日爆文」       | 在规律分析后引导订阅（需产品侧已开通）   |

### 输出示例

完成查询后，你将先看到**完整榜单**（含数据说明、概览与详情），随后可获得**爆款规律分析**摘要，并生成**可视化榜单页**（支持一键导出 PDF），便于团队分享与离线阅读。
