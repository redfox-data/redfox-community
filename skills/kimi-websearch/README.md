# Kimi WebSearch / kimi-websearch

---

## 简介

Kimi WebSearch 是一款联网智能搜索工具，只需用自然语言描述你想了解的话题，就能调用 Kimi 的联网能力为你找到最新、最全的答案。

**核心价值**

- 输入任意问题，自动联网搜索并返回结构化答案
- 无需等待，后台自动轮询结果，搜完即时呈现
- 结果包含完整回答和引用来源，方便核实

**适用对象**

- 🔍 **信息搜集者** — 需要快速获取某个话题的实时信息
- ✍️ **内容创作者** — 为写作寻找参考素材和最新数据
- 🎯 **日常用户** — 想问什么直接问，省去自己翻网页的麻烦

---

## 功能特性

### 核心功能

- 自然语言提问，智能联网搜索最新信息
- 自动等待搜索结果，中途实时提示进度
- 返回结构化答案，附带引用来源链接
- 覆盖 API Key 缺失、超时等异常情况的友好提示

---

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github)（`https://redfox.hk`）提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。
