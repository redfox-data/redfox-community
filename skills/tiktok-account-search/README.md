# TikTok账号搜索 / tiktok-account-search

---

## 简介

输入关键词就能搜到 TikTok 上的博主账号，按粉丝数从高到低给你排好，昵称、粉丝量、个人简介一目了然，点一下链接就能跳转主页看详情。搜不到内容时会自动推荐 10 个相近关键词，帮你换着方向继续找。

**核心价值**

- **内容新鲜不滞后**：每次搜索都直接拉取平台当前的最新账号数据，不会拿旧缓存糊弄你。
- **按粉丝数高低排好**：搜索结果自动按粉丝量从高到低排列，头部大号一眼就能看到。
- **搜不到自动扩词**：关键词搜不出结果时，工具会一口气给你 10 个相近的推荐词，帮你换个角度继续挖。
- **支持翻页浏览**：结果多的时候可以一页页翻着看，每页 10 条，不遗漏任何一个账号。

**适用对象**

- 🔍 **内容创作者** — 找同赛道海外博主，研究对标账号的内容方向和运营思路。
- 📊 **运营 / 数据分析** — 快速摸清某个品类在 TikTok 上有哪些头部账号，掌握竞争格局。
- 🏢 **品牌 / MCN** — 筛选潜在合作博主，按粉丝量评估账号价值，辅助签约决策。
- 🌏 **跨境电商** — 搜索海外细分市场的达人资源，找到匹配的带货 / 推广人选。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入任意关键词搜索 TikTok 账号，按粉丝数降序展示。
- **粉丝排序**：结果按粉丝量从高到低自动排列，头部账号一目了然。
- **主页直达**：每条结果附可点击的主页链接，一键跳转查看账号详情。
- **翻页浏览**：每页展示 10 条结果，支持多页翻看。
- **智能扩词**：搜不到结果时自动生成 10 个相近关键词推荐。

### 特色亮点

- **越搜越准**：翻页时自动带上下游标，结果稳不重复不遗漏。
- **兜底不空军**：任何关键词搜不到都有推荐词帮你兜底，换方向继续挖。

## 使用指南

直接用自然语言说出你想查的博主类型即可，无需记忆固定命令。

### 常用说法速查

| 意图             | 示例话术                           | 效果                                |
| ---------------- | ---------------------------------- | ----------------------------------- |
| 搜某个品类博主   | "看看 TikTok 美妆博主"             | 以"美妆博主"查询，按粉丝数排序      |
| 口语化随口搜     | "海外有什么厉害的健身达人"         | 自动提取"健身达人"，按粉丝数排序      |
| 翻页查看更多     | 表格展示后回复"下一页"             | 自动带上下游标翻页，结果连续不重复     |
| 搜不到换词       | 无结果时查看推荐词，选一个重搜     | 利用 10 个扩词推荐继续探索            |
