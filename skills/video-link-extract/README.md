# 短视频链接提文案 / video-link-extract

---

## 简介

只需一条视频链接，就能自动获取视频里说的每一句话。无论是抖音热门视频的台词、小红书教程的讲解内容，还是快手直播的口播文案，贴个链接就能拿到完整的文字版，还附带每句话出现的时间点。

**核心价值**

- **一条链接出全文**：粘贴抖音、快手、小红书的视频分享链接，自动返回完整文案，无需下载视频或安装任何转写工具。
- **毫秒级时间轴**：每句话精确标注起止时间，直接用于字幕制作和二创剪辑。
- **智能进度追踪**：提交后自动等待处理完成，不用守着刷新，结果出来后即时推送。
- **清晰错误反馈**：链接无效、平台不支持、视频超长等问题一目了然，附带解决建议。

**适用对象**

- 📝 **内容二创作者** — 快速提取爆款视频文案，拆解话术结构，提升二次创作效率。
- 📊 **运营人员** — 分析竞品视频脚本，沉淀可复用的内容框架。
- ✂️ **视频剪辑师** — 一键获取带时间轴的字幕文本，省去手动听写。

---

## 功能特性

### 核心功能

- **多平台覆盖**：自动识别抖音、快手、小红书三大平台的分享链接，一个工具搞定主流短视频。
- **语音转文字**：智能识别视频中的语音内容，方言和嘈杂环境也能稳定输出，整体准确率超 95%。
- **逐句时间标注**：每句话附带精确的起止时间，快速定位视频中的任意片段。
- **自动进度追踪**：提交任务后无需手动刷新，系统自动轮询等待，完成后即时返回结果。
- **批量处理**：支持一次提交多个链接，依次处理并汇总展示每个链接的提取结果。

### 特色亮点

- **零门槛操作**：粘贴链接即可获取文案，不需要下载视频、安装转写软件。
- **毫秒级时间轴**：每句话精确标注起止时间，直接用于字幕制作。
- **自动进度追踪**：提交后无需守着等，智能轮询自动获取最终结果。
- **完整错误说明**：链接无效、平台不支持等问题一目了然，附带解决建议。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 提取视频文案 | "帮我把这个抖音视频的文案提取出来" | 输入链接，自动返回完整文字版 |
| 获取带时间轴的字幕 | "提取这个视频的文案，要带时间戳" | 输出每句详情（含起止时间），可直接做字幕 |
| 分析视频话术 | "分析这个爆款视频说了什么" | 获取完整文案，便于拆解话术结构 |
| 批量提取 | "把这些链接的视频文案都提取出来" | 依次处理多个链接，逐一返回结果 |

### 输出示例

提交链接后，你将收到三部分内容：

**完整文案** — 按语义分段的视频文字内容，直接在对话中展示。

> 大家好，今天我们来聊一聊如何拍出高质量的短视频。首先第一点，光线非常重要。自然光是最好的光源，尽量在窗边拍摄...

**视频内容总结** — 1～3 句话概括视频主题与核心内容，帮你快速了解视频讲了什么。

**每句详情展示** — 生成为独立文件，点击即可查看全部内容的逐句时间标注表格。
