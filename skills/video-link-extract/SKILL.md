# 短视频链接提文案

## 📝 简介

短视频链接提文案让你只需一条视频链接，就能自动获取视频里说的每一句话。无论是抖音热门视频的台词、小红书教程的讲解内容，还是快手直播的口播文案，贴个链接就能拿到完整的文字版，还附带每句话出现的时间点。

适合做内容二创的博主拆解爆款话术、运营人员分析竞品视频脚本、视频剪辑师快速提取字幕素材。

## ✨ 功能特性

### 核心功能

| 功能模块 | 能力描述 | 核心价值 |
|---------|---------|----------|
| 多平台支持 | 自动识别抖音、快手、小红书等平台的视频分享链接 | 一个工具覆盖主流短视频平台，无需来回切换 |
| 智能识别 | AI 精准识别视频中的语音内容并转为文字 | 识别准确率高，方言和嘈杂环境也能稳定输出 |
| 分句 | 每句话标注起止时间，精确到毫秒 | 快速定位某句话在视频中的位置，便于引用和剪辑 |
| 异步等待 | 提交后自动追踪处理进度，无需手动刷新 | 后台自动处理，完成后即时返回结果 |
| 错误提示 | 遇到不支持的平台、超长视频等问题时给出明确提示 | 快速定位问题原因，不用猜测失败原因 |

### 特色亮点

- **零门槛操作**：粘贴链接即可获取文案，不需要下载视频、安装转写软件
- **毫秒级时间轴**：每句话精确标注起止时间，直接用于字幕制作
- **自动进度追踪**：提交后无需守着等，智能轮询自动获取最终结果
- **完整错误说明**：链接无效、平台不支持等问题一目了然，附带解决建议

### 📱 各平台视频链接怎么获取

使用本技能前，需要先从各平台 App 复制视频链接。以下是三大平台的具体操作方式：

#### 抖音

**如何获取链接：**
1. 打开抖音 App，找到你要提取文案的视频
2. 点击右下角 **「分享」** 按钮（箭头图标）
3. 在分享面板中选择 **「复制链接」**

**支持的链接格式：**
- 分享短链：`https://v.douyin.com/xxxxxx/`
- 网页链接：`https://www.douyin.com/video/xxxxxx`

**平台限制：**
- ⚠️ 仅支持**公开视频**，私密或已删除的视频无法提取
- ⚠️ 图文类作品（无视频流）不支持
- ⚠️ 视频时长不能超过 10 分钟

#### 快手

**如何获取链接：**
1. 打开快手 App，找到你要提取文案的视频
2. 点击右下角 **「分享」** 按钮
3. 在分享面板中选择 **「复制链接」**

**支持的链接格式：**
- 分享短链：`https://v.kuaishou.com/xxxxxx`
- 网页链接：`https://www.kuaishou.com/short-video/xxxxxx`

**平台限制：**
- ⚠️ 仅支持**公开视频**，私密或已删除的视频无法提取
- ⚠️ 图文类作品（无视频流）不支持
- ⚠️ 视频时长不能超过 10 分钟

#### 小红书

**如何获取链接：**
1. 打开小红书 App，找到你要提取文案的视频笔记
2. 点击右上角 **「分享」** 按钮（箭头图标）
3. 在分享面板中选择 **「复制链接」**

**支持的链接格式：**
- 网页链接：`https://www.xiaohongshu.com/discovery/item/xxxxxx`
- 分享短链：小红书 App 内复制出的短链接

**平台限制：**
- ⚠️ 仅支持**视频笔记**，图文笔记（纯图片/文字）无视频可提取
- ⚠️ 笔记需为**公开状态**，私密笔记无法访问
- ⚠️ 视频时长不能超过 10 分钟

> 💡 **小提示：** 如果不确定复制出来的链接能不能用，直接粘贴给我就行，我会帮你检查。

## 🔄 工作流程

1. **输入视频链接**：提供抖音、快手、小红书等平台的视频分享链接
2. **提交处理任务**：系统自动提交文案提取任务，返回任务标识
3. **智能等待处理**：自动追踪任务进度，间隔几秒检查一次，直到完成
4. **返回提取结果**：输出分段完整文案 + 视频内容总结，每句详情生成为独立 `.md` 文件供点击查看

> 完整执行流程详见 `references/core_workflow.md`

## 🔗 接口文档

### 接口地址

| 接口名称 | 请求方式 | 接口地址 |
|---------|---------|----------|
| 提交文案提取任务 | POST | https://redfox.hk/story/api/parseWork/audioTextExtract/submit |
| 查询提取结果 | POST | https://redfox.hk/story/api/parseWork/audioTextExtract/result |

### 提交任务 - 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| url | string | 是 | 视频链接（支持抖音/快手/小红书分享链接或网页链接） |
| source | string | 是 | 接口来源标识，固定值：`"短视频链接提文案-GitHub"` |

### 提交任务 - 响应结构

| 字段名 | 类型 | 说明 |
|---------|------|------|
| taskId | string | 任务标识，用于后续查询提取结果 |

### 查询结果 - 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| taskId | string | 是 | 提交任务时返回的任务标识 |
| source | string | 是 | 接口来源标识，固定值：`"短视频链接提文案-GitHub"` |

### 查询结果 - 响应结构

| 字段名 | 类型 | 说明 |
|---------|------|------|
| taskId | string | 任务标识 |
| status | string | 任务状态：queued（排队中）/ running（处理中）/ succeeded（成功）/ failed（失败） |
| text | string | 提取的完整文案内容（仅成功时有值） |
| stampSents | list | 每句详情列表（仅成功时有值），每项含 textSeg（文本片段）、start（开始时间ms）、end（结束时间ms） |
| failReason | string | 失败原因（仅失败时有值） |

### 数据范围

| 数据类型 | 获取范围 | 说明 |
|---------|---------|------|
| 平台支持 | 抖音、快手、小红书 | 需提供对应平台的视频分享链接或网页链接 |
| 视频时长 | 不超过 10 分钟 | 超长视频请先剪辑后再提取 |
| 内容类型 | 仅支持视频作品 | 图文类型作品无法提取 |
| 视频可见性 | 公开视频 | 私密或已删除的视频无法提取 |
| 返回精度 | 毫秒级时间戳 | 每句文案标注精确起止时间 |

### 各平台特殊说明

| 平台 | 链接获取位置 | 常见问题 |
|------|------------|---------|
| 抖音 | App 内视频页 → 分享按钮 → 复制链接 | 私密账号发布的视频无法提取；部分直播间回放可能不支持 |
| 快手 | App 内视频页 → 分享按钮 → 复制链接 | 同抖音，仅支持公开视频 |
| 小红书 | App 内笔记页 → 右上角分享 → 复制链接 | 需确认笔记为视频类型（封面有播放图标）；图文笔记不支持 |

### 降级处理

当未找到目标数据或处理失败时：
1. 链接解析失败 → 提示用户检查链接是否有效、是否来自支持的平台
2. 视频超时 → 提示用户视频需在 10 分钟以内，建议先裁剪
3. 图文类型 → 提示该链接为图文作品，无视频可提取
4. 系统繁忙 → 提示稍后重试，系统自动恢复

> ⛔ **重要：** 当 API 返回异常或提取失败时，直接向用户展示 API 原始错误信息，**不得**通过浏览器或其他方式访问链接内容作为回退方案。
>
> 💰 **积分退回：** 提取失败时，同时告知用户扣除的积分已退回当前账号，可前往 **积分记录** 查看明细。

## ❓ 常见问答

### 使用相关

**Q：支持哪些平台的视频？**
A：目前支持抖音、快手、小红书三大平台的视频链接。直接粘贴分享链接或网页链接即可。

**Q：视频最长可以多久？**
A：支持最长 10 分钟的视频。如果视频超过 10 分钟，建议先用剪辑工具裁剪后再提取。

**Q：提取结果包含什么？**
A：包含三部分 —— 按语义分段的完整文案、视频内容简要总结在对话中直接展示，每句详情（**分**秒格式起止时间，全部列出）生成为独立 `.md` 文件，点击即可查看完整内容。

**Q：图文类型的内容能提取吗？**
A：不能。本技能专门提取视频中的语音内容，图文作品没有视频流，无法处理。

### 故障排除

**Q：提示"链接解析失败"怎么办？**
A：请确认链接来自抖音、快手或小红书，且是视频类型（非图文）。分享链接和网页链接都支持。

**Q：等待很久没有返回结果？**
A：视频处理需要一定时间，系统会自动等待并追踪进度。如果超过 2 分钟仍无结果，可以重新提交尝试。

**Q：提取的文案有错别字怎么办？**
A：语音识别在嘈杂环境或方言场景下可能出现少量偏差，建议人工校对关键内容。整体准确率在 95% 以上。
