# 全网热点追踪 /trending-hub

## 简介

帮你把抖音、微博、B站、快手、知乎、头条、百度等7个平台的热搜聚合在一起，省得一个个平台去看。

**核心价值**

一键获取7大平台热搜数据，小时级更新。按平台分类独立展示，每个平台最多50条热搜，支持按关键词/平台筛选，告别逐平台查看的痛苦。

**适用对象**

- 🔍 竞品监控 — 查看特定平台热搜，了解竞争对手动态
- 🎯 选题调研 — 按关键词筛选相关热搜，快速匹配选题方向
- 📊 数据分析 — 获取全平台原始热搜数据，进行深度分析
- 📰 舆情追踪 — 监控特定事件在各平台的热搜表现 ---

## 核心功能


- **全平台热搜展示**：7大平台热搜独立展示，每个平台最多50条
- **多平台组合查询**：支持查询单个和多个指定平台热搜
- **关键词精准筛选**：支持按关键词筛选相关热搜
- **多时间范围查询**：支持实时、今日、昨日、本周等多种时间范围
- **完整榜单查看**：每个平台可查看全部50条热搜
- **订阅推送服务**：定时推送最新热榜/昨日热榜

## 前置条件

### 注册红狐 hub 账号获取 REDFOX_API_KEY

- 获取 REDFOX_API_KEY（前往 [红狐 hub](https://redfox.hk/dashboard/keys?souce=github) 申请）

### 环境变量配置

| 变量名 | 必填 | 说明 |
| --- | --- | --- |
| `REDFOX_API_KEY` | 是 | API 访问密钥 |

**macOS（zsh）**

在 `~/.zshrc` 末尾添加一行（将引号内换成你的密钥）：

```bash
export REDFOX_API_KEY="your_api_key_here"
```

保存后执行：

```bash
source ~/.zshrc
```

**Windows（PowerShell）**

- **仅本次终端有效**：执行后立刻生效，**无需再跑别的命令**；关掉窗口后失效。

```powershell
$env:REDFOX_API_KEY = "your_api_key_here"
```

- **写入用户环境变量（持久）**：执行 `setx` 后，**当前这个 PowerShell 窗口里仍然没有该变量**，需要 **关闭并重新打开** 终端（或重启 Cursor / VS Code 等），新开的窗口里才会读到 `REDFOX_API_KEY`。

```powershell
setx REDFOX_API_KEY "your_api_key_here"
```
