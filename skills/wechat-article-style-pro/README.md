# 公众号文案改写 / wechat-article-style-pro

---

## 简介

本 Skill 面向 **公众号爆款文风改写**：把你提供的草稿当作「自己的原文」，在对话中直接输出**已改写完成**的公众号风格**标题 + 正文**（不说「帮你改好了」之类旁观者口吻），让内容更贴近高阅读公众号的常见表达。

**核心价值**

- **爆款语感**：标题直白、有钩子（一般不超过 **20 字**），正文口语化、态度鲜明，兼顾痛点与共鸣。  
- **结构可感知**：参考爆款常见写法——开头抓人、分点展开、结尾引导互动，并附带 **5～10 个**话题标签。  
- **关键词融入**：正文中自然带上 **2～3 个**公众号常见类关键词（如指南、教程、秘籍、经验分享等，见包内规则）。  
- **轻量排版**：分段清晰，每段可点缀少量 Emoji，短句为主。  

**适用对象**

公众号运营、编辑、自媒体作者，以及需要把通用稿、活动稿、提纲快速改成「更像公众号」文本的内容创作者。

**运行环境**：**Python 3**；改写前须在本技能包根目录运行 **`scripts/rewrite.py`**（`report` 子命令拉取规则参考；`prompt` 仅本地查看摘要）。在已启用本 Skill 的 Agent 中通过自然语言触发。

---

## 功能特性

### 核心功能

- **输入即原文**：你给出的文字视为待发布底稿，输出为改写后的成稿，而非修改建议清单。  
- **标题 + 正文一次给齐**：不反复追问，按规则直接交付。  
- **规则驱动**：改写前执行 **`python scripts/rewrite.py report`**，再结合包内行文原则与风格要点生成内容。  

### 特色亮点

- **金句与对比**：观点尽量一针见血，善用对比、反问与排比增强节奏。  
- **读者视角**：强调「与我有关」「对我有用」，避免空泛说教。  
- **可传播**：核心句尽量好记、好转发。

## 使用指南

### 你可以怎么说

| 你的说法 | 大致会得到什么 |
| --- | --- |
| 「把下面这段改成公众号风格」+ 粘贴正文 | 先跑规则脚本，再输出新标题与全文 |
| 「公众号爆款标题 + 正文，主题：……」 | 按主题组织成稿，标题控制在 20 字内 |
| 「更犀利一点 / 更口语」 | 在爆款文风约束下加强语气与互动感 |

### 使用提示

- 请**一次性提供**待改写正文或清晰主题，便于直接成稿。  
- 输出以**作者身份**撰写，勿期待「修改意见」式回复。  
- 若需核对标题字数、关键词数量、标签与 Emoji 等，以当次成稿自检为准。

## 注意事项与限制

- 本 Skill 提供**文风改写**，不保证阅读数据；发布前请自行核对事实与合规。  
- 改写依赖脚本返回的规则参考；**请勿**在未执行脚本流程时要求「凭空按爆款规律写」。  
- 不得捏造数据、案例或他人引述；你提供的素材须可负责。  
- 遵守微信公众平台内容规范，避免违规导流与夸大宣传。
