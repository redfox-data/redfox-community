# 公众号爆款文案改写 / wechat-article-style

---

## 简介

把任意草稿改写成高阅读潜力的公众号爆款文案——标题有钩子、正文有态度，并自然融入公众号特色表达。

**核心价值**

- **风格一键转化**：以发起者视角输出，保留原文段落结构，整体切换为公众号口语化、犀利直接的文风
- **爆款结构对齐**：标题控制在 20 字内，正文参考爆款开头钩子、分点结构与结尾互动引导
- **特色词汇融入**：从指南、教程、秘籍等高频词中选取 2～3 个，自然写入正文，增强平台辨识度
- **可读可传播**：金句总结、适度 Emoji、结尾 Tags，方便读者阅读与转发

**适用对象**

- ✍️ 公众号运营 — 快速把素材改写成可发布的爆款口吻文章
- 📱 自媒体创作者 — 将笔记、演讲稿等改写成更适合微信阅读的表达
- 🏢 品牌 / 市场 — 统一对外稿件的公众号风格，提升打开率与互动

---

## 功能特性

### 核心功能

- **文案风格改写**：输入你的原文，输出公众号风格的标题 + 正文（以作者身份直接给出成稿，无需反复确认）
- **爆款标题**：数字型、疑问句、感叹句等钩子写法，标题不超过 20 字
- **正文结构优化**：痛点共鸣、分点说明、对比展示；结尾引导评论互动，附带 5～10 个相关 Tags
- **行文原则内置**：强调「跟我有关、对我有用」、观点鲜明、情感共鸣，避免枯燥说教

## 使用指南

直接粘贴或描述你要改写的文案即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 改写一篇成稿 | 「帮我把下面这段改成公众号爆款风格：[你的原文]」 | 输出标题 + 正文，含互动引导与 Tags |
| 只要标题 | 「给这段内容起一个 20 字内的公众号爆款标题」 | 给出符合钩子规律的标题 |
| 强化观点 | 「改写得再犀利一点，多加金句」 | 在爆款结构下加强态度与传播点 |
| 指定语气 | 「口语化一些，像跟读者聊天」 | 亲切自然、短句为主的公众号口吻 |

## 使用场景

| 场景 | 角色 | 示例问法 | 收益 |
| ---- | ---- | -------- | ---- |
| 日常推文改稿 | 公众号编辑 | 「这篇产品介绍太书面了，改成公众号能发的版本」 | 缩短从素材到可发布稿的时间 |
| 多平台内容复用 | 自媒体博主 | 「把这篇小红书笔记改写成公众号文章」 | 同一主题适配微信阅读习惯 |
| 活动 / 促销文案 | 品牌运营 | 「把活动通知改得更有紧迫感、更像爆款推文」 | 提升打开意愿与转发传播 |
| 知识类长文 | 内容创作者 | 「把干货大纲扩写成有钩子的公众号正文」 | 结构清晰、可读性更强 |
