# 公众号账号搜索 / wechat-account-search

---

## 简介

输入关键词就能搜到微信公众号博主账号，账号名称、账号ID、账号简介一览无余。

**核心价值**

- **内容新鲜不滞后**：每次搜索都直接拉取平台当前的最新账号数据，不会拿旧缓存糊弄你。
- **信息维度实用**：展示账号名称、账号ID、账号简介，快速了解账号定位和内容方向。
- **支持翻页浏览**：结果多的时候可以一页页翻着看，不遗漏任何一个账号。

**适用对象**

- 🔍 **内容创作者** — 找同领域公众号，研究对标账号的内容方向和运营思路。
- 📊 **运营 / 数据分析** — 快速摸清某个品类在公众号上有哪些头部账号，掌握竞争格局。
- 🏢 **品牌 / MCN** — 筛选潜在合作账号，按领域评估账号价值，辅助合作决策。
- 📝 **文案策划** — 搜索细分赛道的公众号，寻找选题灵感和内容参考。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入任意关键词搜索微信公众号博主账号。
- **数据展示**：展示账号名称、账号ID、账号简介，快速了解账号定位和内容方向。
- **翻页浏览**：每页展示多条结果，支持多页翻看。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) 提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言说出你想查的公众号类型即可，无需记忆固定命令。

### 常用说法速查

| 意图             | 示例话术                           | 效果                                |
| ---------------- | ---------------------------------- | ----------------------------------- |
| 搜某个领域公众号 | "看看财经公众号"                   | 以"财经"查询，按接口返回顺序展示      |
| 口语化随口搜     | "有什么厉害的科技类公众号"         | 自动提取"科技"查询      |
| 翻页查看更多     | 表格展示后回复"下一页"             | 自动递增页码翻页，结果连续不重复     |
