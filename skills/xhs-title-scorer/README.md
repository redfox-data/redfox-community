# 小红书爆款标题打分器 / xhs-title-scorer

---

## 简介

基于小红书真实爆款数据，对你的标题进行 6 维度加权评分，并给出可落地的修改建议，帮你在发布前判断标题的流量潜力。

**核心价值**

- 用真实爆款数据说话，评分依据可追溯，不凭感觉打分
- 6 维度拆解标题短板，精准定位影响流量的关键问题
- 提供多风格修改版本，直接可用，无需二次构思

**适用对象**

- 📝 **小红书博主** — 发布前快速验证标题，减少低流量笔记
- 🛍️ **品牌 / 电商运营** — A/B 测试前筛掉明显弱势标题
- 🏢 **MCN / 内容策划** — 批量评估选题标题，提升内容投放效率

---

## 功能特性

### 核心功能

- **爆款数据驱动评分**：基于同赛道近期真实爆款，提炼结构、用词、情绪等规律，再对标你的标题打分
- **6 维度加权评分**：主题匹配度、结构合规度、利益清晰度、情绪唤醒度、稀缺性感知、合规安全性，按权重计算综合得分（满分 10 分）
- **等级判定**：S / A / B / C 四级，对应不同优化策略
- **爆款重合识别**：若你的标题与已有爆款高度一致，直接标注并给予 S 级认定
- **多风格修改建议**：A/B/C 级结果附带核心问题清单、参考爆款标题及干货风 / 情绪风 / 猎奇风修改版本

## 使用指南

直接用自然语言描述你的标题和相关主题即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 评分单个标题 | 「帮我评分'这款防晒霜真的绝了'，关键词防晒霜」 | 查询防晒霜爆款数据，输出 6 维度评分报告与优化建议 |
| 只给标题不给关键词 | 「'新手必看的护肤技巧' 这个标题如何」 | 自动从标题提取关键词，完成评分 |
| 指定主题方向 | 「评分'夏天这样穿超好看'，主题是夏日穿搭」 | 按夏日穿搭赛道爆款规律评分 |

### 输出示例

完成评分后，你将收到以下内容：

**评分报告**（S / A / B / C 级 + 综合得分 + 6 维度明细表）

**优化建议**（A/B/C 级时输出）：
- 核心问题清单
- 3 ～ 5 条同主题参考爆款标题（含作者与互动数）
- 2 ～ 3 个修改版本（干货风 / 情绪风 / 猎奇风）
