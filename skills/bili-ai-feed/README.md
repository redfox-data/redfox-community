# B站AI信息源 / bili-ai-feed

---

## 简介

每日自动扫描B站AI创作内容，按点赞量筛选爆款视频，智能聚类后生成包含封面图与互动数据的可视化日报；同步基于热门话题执行AI情报调查，多引擎搜索与交叉验证，输出结构化调查报告。

**核心价值**

- 精准定位B站AI赛道每日高热度视频，告别人工刷屏
- 智能聚类自动归并话题方向，一眼看清当日AI内容分布
- 多引擎交叉验证情报调查，追踪话题背后的产品、人物与舆情信息
- 一键订阅，日报自动攒在本地文件夹，无需手动操作

**适用对象**

- 🔍 AI内容创作者 — 快速获取B站AI热点选题，把握创作方向
- 📊 AI行业观察者 — 追踪AI赛道每日爆款趋势与话题演变
- 🛠️ AI产品经理 — 发掘竞品动态、用户口碑与行业信号
- 🎯 运营与增长团队 — 发现高互动内容模式，优化投放策略

---

## 功能特性

### 核心功能

- **爆款发现** — 从B站AI相关视频中按点赞量筛选热门内容，精准定位高互动视频
- **智能聚类** — 自动识别话题方向（AI教程、大模型、AI绘画等），分类由当日内容动态决定
- **AI情报调查** — 基于聚类热门话题，执行竞品情报、舆情事件、人物背景、信息交叉验证四类调查
- **可视化日报** — 深色主题HTML页面，包含封面图、互动数据与视频直达链接
- **一键订阅** — 通过订阅命令开启每日自动产出，日报自动保存在本地文件夹

### 特色亮点

- 🌐 **16+搜索引擎协同** — 覆盖百度、Google、DuckDuckGo、微信搜狗、头条等，按调查目标智能编排引擎组合
- 🔍 **四类调查模式** — 竞品情报、舆情事件、人物背景、信息交叉验证，满足不同情报需求
- 📊 **可信度标注体系** — 信息源分ABCD四级，结果标注已确认/待确认/已否定/单一来源
- 🤖 **全自动工作流** — 扫描→聚类→调查→输出，一次触发即可获得日报+情报报告

- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 获取今日日报 | "帮我生成今天的B站AI日报" | 扫描AI相关视频，聚类后生成HTML日报并自动打开 |
| 关注特定方向 | "帮我看看B站AI绘画和ChatGPT相关的爆款" | 聚焦指定关键词方向的视频内容 |
| 查看历史日报 | "帮我生成6月10号的B站AI日报" | 回溯指定日期的AI视频数据 |
| 开启每日订阅 | "帮我开启B站AI日报的每日订阅" | 每天自动产出日报，保存在本地文件夹 |
| 执行情报调查 | "帮我调查一下B站最近热门的那个AI产品" | 基于日报话题自动执行多引擎交叉验证调查 |

### 输出示例

日报运行后自动生成以下内容：

- **HTML可视化日报** — 深色主题页面，含封面图、点赞/播放数据、视频直链，保存在 `~/Downloads/QoderReports/`
- **终端分类表格** — 按话题分类展示视频汇总，标注占比与亮点
- **AI情报调查报告** — 基于TOP3热门话题的结构化报告，包含维度发现、来源与可信度标注
