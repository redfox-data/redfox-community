# AI-B站信息源 / bili-ai-feed

---

## 简介

AI 领域 B 站内容每日自动扫描工具，从 B 站 AI 相关视频中按点赞量发现爆款内容，智能聚类话题后生成带封面图与互动数据的 HTML 日报，并基于热门话题自动执行多引擎 AI 情报调查，输出结构化调查报告。内置公共访问密钥，约 10,000 次免费额度，开箱即用。

**核心价值**

- **爆款发现 + 智能聚类**：每日扫描 B 站 AI 视频，按点赞量筛选热门内容，自动归类话题方向（AI 教程、大模型、AI 绘画等），分类由当天内容动态决定。
- **可视化 HTML 日报**：深色主题日报，包含视频封面图、互动数据（点赞/评论/转发）、视频直链，保存在本地文件夹，自动浏览器打开。
- **AI 情报调查联动**：日报生成后自动基于 TOP 3 热门话题执行多引擎情报调查（竞品/舆情/人物/验证），输出带可信度标注的结构化报告。
- **一键订阅自动化**：`--subscribe` 开启每日自动产出，日报自动攒在本地文件夹，无需手动触发。

**适用对象**

- 🤖 **AI 从业者 / 研究者** — 每日掌握 B 站 AI 领域爆款内容和话题趋势。
- 📊 **内容运营 / 自媒体** — 发现 AI 赛道热点，获取选题灵感与内容方向。
- 📈 **产品经理 / 创业者** — 了解 AI 产品在 B 站的用户反馈和讨论热度。
- 🔍 **情报分析师** — 基于 B 站热点联动多引擎交叉验证，获取可追溯的结构化情报。

## 功能特性

### 核心功能

- **爆款发现**：从 B 站 AI 相关视频中按点赞量筛选热门内容，精准定位高热度 AI 视频。
- **智能聚类**：自动识别话题方向（AI 教程、大模型、AI 绘画等），每天分类由内容动态决定。
- **AI 情报调查**：基于热门话题自动执行多引擎搜索 + 交叉验证，深度挖掘话题背后的情报。
- **可视化日报**：深色主题 HTML，封面图 + 互动数据 + 视频直链，直观展示每日 AI 热点。
- **一键订阅**：`--subscribe` 开启每日自动产出，日报自动攒在本地文件夹。
- **记录保存**：调查报告自动保存至红狐平台，支持历史回溯与趋势分析。

### 特色亮点

- **日报 + 情报双输出**：不只展示爆款视频，还基于聚类话题联动 AI 情报调查，生成带可信度标注的结构化报告。
- **话题自动匹配调查模式**：TOP 话题自动匹配竞品/舆情/人物/验证四种调查模式，选择最优引擎组合执行三轮搜索。
- **开箱即用**：内置公共访问密钥，约 10,000 次免费额度，无需额外配置即可体验完整功能。
- **订阅式自动化**：一键订阅后每日自动产出日报，结果自动攒入本地文件夹，完全自动化运行。

## 密钥获取与安全说明

- 本技能内置公共 `REDFOX_API_KEY`（约 10,000 次免费额度），也可使用自有密钥。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能（如需使用自有密钥）。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言触发，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 生成今日日报 | 「帮我生成今天的 AI-B站日报」 | 扫描 B 站 AI 视频 → 聚类 → 生成 HTML 日报 + 情报调查报告 |
| 自定义关注方向 | 「帮我关注 AI 教程、AI 绘画方向的 B 站内容」 | 按指定关键词筛选爆款视频并生成日报 |
| 开启每日订阅 | 「帮我开启 AI-B站日报每日订阅」 | 每日自动产出，日报攒在本地文件夹 |
| 查看历史日报 | 「帮我看看 2026-06-10 的 AI-B站日报」 | 回看指定日期的日报内容 |

### 输出示例

每次运行后，你会收到终端输出 + HTML 日报（浏览器自动打开）：

**分类概览表格**：扫描 N 条热门视频，聚类 M 个分类，展示各分类数量、占比和头部亮点

**AI 情报调查报告**（基于 TOP 3 话题）：
- 🔥 新兴起量信号：低条数但高互动的话题方向
- 👤 核心达人：当日产出最多/获赞最高的 B 站创作者
- 📊 话题调查报告：调查模式 + 引擎组合 + 维度发现表格（带 A~D 级可信度标注）
- 🔗 跨平台对比建议：建议同步关注的抖音、小红书、公众号同话题动态

**HTML 日报**：深色主题，视频封面图 + 互动数据 + 跳转链接，保存在 `~/Downloads/QoderReports/`

## 重要数据说明

- 每日扫描约 200 条 AI 相关 B 站视频，按点赞量筛选爆款，智能聚类话题分类由当天内容动态决定。
- HTML 日报保存在 `~/Downloads/QoderReports/` 目录，自动浏览器打开。
- 情报调查基于多搜索引擎交叉验证，关键信息至少 2 个独立来源确认后才标注为「已确认」。
- 信源分为 ABCD 四级：A 级（官方/权威媒体）、B 级（行业媒体/专业平台）、C 级（社交媒体/自媒体）、D 级（匿名/未验证来源）。
- 内置公共密钥约 10,000 次免费额度，超出后可配置自有 `REDFOX_API_KEY`。
- 所有数据均来源于公开可访问平台，不涉及非公开数据采集。
