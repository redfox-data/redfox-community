# 微博账号作品回采 / weibo-post-search

---

## 简介

输入微博主页链接或 uid 就能查到该用户刚发布的博文，转发、评论、点赞、发布时间一目了然，点内容直接跳转原博，给到你的都是平台最新发布的内容，不会拿旧缓存糊弄你。

**核心价值**

- **内容新鲜不滞后**：每次查询直接拉取平台最新博文，只给你刚发布的内容，不是过期存档。
- **数据一眼看透**：转发、评论、点赞、发布时间四项指标并列展示，互动热度一表看清。
- **翻页随心浏览**：海量博文支持分页翻阅，一句「下一页」继续看更多历史内容。

**适用对象**

- 🔍 **内容运营 / 竞品分析** — 快速查看目标博主近期产出，掌握更新节奏
- 📊 **舆情监测** — 追踪特定用户动态，第一时间发现关键发声
- 🎯 **创作者研究** — 了解同类博主内容风格和互动水平

---

## 功能特性

### 核心功能

- **用户查询**：输入微博主页链接或 uid，查询该用户最新博文。
- **分页浏览**：支持多页结果浏览，按需查看更多历史博文。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) 提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 查用户博文 | 「查下 https://weibo.com/1784473157 的博文」 | 提取 uid 后查询并展示结果 |
| 直接给 uid | 「查 1784473157 的最新微博」 | 直接用 uid 查询 |
| 翻页查看 | 「下一页」/ 「上一页」 | 切换当前结果页码 |
