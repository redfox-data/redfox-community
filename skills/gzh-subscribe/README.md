# 公众号文章订阅 / gzh-subscribe

---

## 简介

订阅竞对、同类和关注公众号，自动抓取每日发文，终端表格 + HTML 日报双模式呈现，内置公共 API Key，开箱即用。

**核心价值**

- **收件箱式订阅**：像订阅 Newsletter 一样订阅公众号，最多 20 个。
- **每日 9 点准时推送**：一键安装定时任务，每天早上自动生成日报。
- **三类分组管理**：竞对账号 / 同类账号 / 关注账号，按目的分组展示。

**适用对象**

- 📊 **内容运营** — 盯梢竞品公众号，了解对手发文动态。
- 📝 **自媒体创作者** — 追踪同类账号，获取选题灵感。
- 🏢 **品牌 / 市场团队** — 监控品牌舆情，关注行业大号。

---

## 功能特性

### 核心功能

- **收件箱式订阅**：像订阅 Newsletter 一样订阅公众号，最多 20 个，名称即可。
- **每日自动推送**：一键安装定时任务，每天 09:00 自动拉取并生成 HTML 日报。
- **三类分组管理**：「竞对账号」盯对手、「同类账号」找灵感、「关注账号」追大神。
- **关键数据一屏尽览**：发文日期、标题、简介、阅读数、点赞数，原文链接一键直达。
- **终端 + 日报双模式**：命令行实时查表，HTML 日报适合分享存档。

## API Key 说明

内置公共 API Key，约 **10000 次免费额度**，零配置直接使用。额度用完后可前往 [RedFoxHub](https://redfox.hk/login?source=github) 注册获取个人 API Token：

| 方式                 | 命令                                                                    |
| -------------------- | ----------------------------------------------------------------------- |
| **环境变量**（推荐） | `export REDFOX_API_KEY=ak_你的密钥`                                     |
| **命令行参数**       | `python3 "$SKILL_PATH/assets/subscribe.py" fetch --api-key ak_你的密钥` |
| **配置文件**         | `echo '{"api_key":"ak_你的密钥"}' > ~/.qoder/apis/redfox.json`          |

## 使用指南

用自然语言描述你想订阅或查看的公众号即可。

### 常用说法速查

| 意图         | 示例话术                         | 效果                  |
| ------------ | -------------------------------- | --------------------- |
| 添加订阅     | 「帮我订阅 XX 公众号」           | 添加到订阅列表        |
| 查看今日发文 | 「今天订阅的公众号都发了什么」   | 终端表格展示今日发文  |
| 生成日报     | 「帮我生成公众号日报」           | 生成 HTML 日报并打开  |
| 开启每日推送 | 「每天早上帮我推送订阅的公众号」 | 安装每日 9 点定时任务 |

## 使用场景

| 场景     | 角色     | 示例问法                   | 收益                  |
| -------- | -------- | -------------------------- | --------------------- |
| 竞品监控 | 内容运营 | 「订阅几个竞品公众号」     | 每日自动获取竞品动态  |
| 灵感追踪 | 自媒体   | 「关注几个同类赛道的账号」 | 发现选题和写作角度    |
| 日常推送 | 品牌团队 | 「开启每日推送」           | 每天 9 点自动收到日报 |

### 注意事项

- 最多订阅 20 个公众号。
- 如需搜索公众号文章（非订阅），请使用 `gzh-search` 技能。
