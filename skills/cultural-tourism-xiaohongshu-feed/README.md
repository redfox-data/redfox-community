# 文旅小红书信息源 / cultural-tourism-xiaohongshu-feed

---

## 简介

搜索全网文旅相关小红书热门笔记，按点赞量筛选最火内容，自动按主题聚类并生成精美的可视化报告，支持时间范围筛选与每日自动订阅。

**核心价值**

- **全量覆盖**：单次获取 200 条文旅笔记，数据充足、覆盖面广。
- **智能分类**：自动按 `type/topic` 标签聚类，一眼看清各赛道内容分布。
- **精美报告**：深色主题 HTML，分区卡片网格 + 封面图展示，支持全文搜索。
- **每日订阅**：定时自动产出日报，无需手动重复操作。

**适用对象**

- 📱 **内容运营 / 新媒体人** — 掌握小红书文旅赛道每日热点，寻找选题灵感。
- 🏨 **景区 / 文旅从业者** — 了解同行动态与游客反馈，辅助营销决策。
- 📊 **文旅研究员** — 追踪文旅话题趋势，获取一手数据素材。

---

## 功能特性

### 核心功能

- **关键词搜索**：支持按景区/城市/主题精准搜索，也支持空关键词获取全量文旅内容。
- **时间筛选**：按日期范围过滤数据，灵活控制数据窗口。
- **自动聚类**：200 条数据按主题标签自动分类，降低信息熵，聚焦重点。
- **可视化报告**：深色主题 HTML 报告，分区卡片网格 + 封面图，精美呈现。
- **每日订阅**：开启 `--subscribe` 后定时自动产出，无需手动重复操作。
