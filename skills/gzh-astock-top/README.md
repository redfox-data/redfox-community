# A股公众号大V / gzh-astock-top

---

## 简介

A股公众号大V工具，输入日期即可获取A股领域头部公众号大V的账号数据与最新文章数据，帮助投资者、内容运营和金融从业者快速掌握A股领域公众号的运营表现和最新内容动向。

**核心价值**

- **一站式榜单**：一键查询49个A股头部公众号的当日数据，覆盖官媒/机构与个人大V两大类别
- **多维数据**：同步获取平均阅读数、红狐指数、文章标题、阅读量、点赞评论等核心指标
- **AI智能概要**：自动为每篇文章生成一句话核心观点，快速把握内容重点
- **订阅追踪**：支持按序号订阅关注的账号，每日只查已订阅列表，减少重复查询

**适用对象**

- 📈 投资者 — 追踪A股领域头部观点，把握市场情绪与热点方向
- 📝 内容运营 — 研究头部公众号的选题策略与数据表现，优化内容方向
- 📊 金融从业者 — 监控行业舆论动态，获取竞品分析参考
- 🔍 数据分析师 — 批量获取公众号运营数据，辅助量化研究与报告

---

## 功能特性

### 核心功能

- **榜单查询**：按日期查询A股领域公众号大V榜单，官媒/机构与个人大V分类展示，每类最多30个账号
- **账号数据**：获取每个账号的平均阅读数、红狐指数、账号描述等核心运营指标
- **当日文章**：获取每个账号在指定日期发布的最新文章，包含标题、链接、阅读数、点赞数、评论数等互动数据
- **AI内容概要**：基于文章标题自动生成一句话核心观点，帮助快速浏览内容
- **订阅推送**：按序号订阅关注的官媒或大V账号，每日一键查询已订阅账号的最新文章更新

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 重要数据说明

- 账号数据来源于红狐API收录的公众号信息，覆盖49个固定的A股头部公众号（19个个人大V + 30个官媒/机构）
- 文章数据为账号在指定日期发布的作品，由服务端直接筛选返回
- 互动数据（阅读数、点赞数、评论数等）截止为数据入库时间，非实时数据
- 红狐平台每日 **07:00** 更新前一日数据，建议 **07:30** 之后查询以获得最新日报
- 仅当日有发布文章的账号才会出现在结果中，因此不同日期的结果账号数量可能不同
