# B站搜账号下作品集 / B站搜账号下作品集

---

## 简介

输入B站UP主UID，实时获取该账号发布的全部视频作品，数据实时更新，支持翻页浏览完整列表。

**核心价值**

- **实时数据**：每次查询均调用实时接口，返回最新作品数据，非缓存或历史记录。
- **快速定位爆款**：作品按播放量降序排列，一眼找到UP主最受欢迎的视频。
- **翻页浏览**：支持逐页翻阅，即使作品数量庞大也能完整查看。

**适用对象**

- 🔍 **内容研究者** — 快速了解任意UP主的内容产出与爆款分布。
- 📊 **数据分析师** — 获取账号作品全量数据，辅助内容趋势研判。
- 🏢 **品牌 / MCN** — 评估KOL内容质量与数据表现，辅助合作决策。

---

## 功能特性

### 核心功能

- **UID查询**：输入B站UP主的UID（纯数字），即可获取该账号全部已发布的视频作品列表，包含播放量、弹幕数、评论数、收藏数等核心数据。
- **播放量排序**：返回结果自动按播放量降序排列，快速定位爆款作品。
- **翻页浏览**：支持逐页浏览更多作品，每页返回 20 条，满足大量作品的查阅需求。
- **完整作品信息**：每条作品包含标题、播放量、弹幕数、评论数、收藏数、时长、发布时间、视频链接及封面图等完整信息。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。
