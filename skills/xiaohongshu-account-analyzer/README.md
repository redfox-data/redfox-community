# 小红书账号诊断 / xiaohongshu-account-analyzer

## 简介

我是深耕小红书账号分析的诊断师，擅长用数据说话，帮你发现账号的真实问题。

**核心价值**

从定位模糊到变现困难，从选题迷茫到更新瓶颈，给你一份基于数据的七维度量化评分（定位、粉丝、选题、封面、爆文、互动、产能）、并对标行业平均水平的诊断报告和可落地的运营优化建议和相似账号推荐

**适用对象**

- 📱 小红书博主 — 诊断账号健康度，发现问题找到优化方向
- 📦 内容运营 — 评估商业价值，制定账号发展规划
- 🏢 品牌方 — 评估达人合作价值，匹配投放候选人
- 🔍 竞品分析 — 对比分析竞品账号，制定竞争策略

## 核心功能

- **七维度评分诊断**：账号定位、粉丝画像、选题体系、封面风格、爆文能力、互动规模、更新产能
- **生命周期分析**：基于数据评估账号所处阶段和发展潜力
- **行动处方生成**：针对问题给出可落地的优化建议
- **相似账号推荐**：推荐 2-5 个可学习的相似账号
- **多账号对比分析**：横向对比多个账号，输出核心差异和各自发展建议
- **HTML 可视化报告**：生成可导出图片的诊断报告

## 重要数据说明

- **账号数据范围**：百万级小红书账号库。少概率存在无账号数据的情况。
- **无账号数据的处理**：搜索时提示用户接收推送，会触发实时采集。30 分钟后自动推送对应账号的诊断报告。
- **分析数据范围**：基于账号数据和近 7 天的作品数据进行分析，每日更新昨日增量数据。
- **数据差距说明**：数据截止为入库时间，不是实时数据，入库后互动数据可能持续增长
