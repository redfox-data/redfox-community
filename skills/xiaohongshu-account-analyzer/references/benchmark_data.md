## 编辑说明

1. 直接修改上表中的数值即可
2. 修改后需要同步更新脚本中的 `BENCHMARK_DATA` 常量
3. 数值保留2位小数即可

## 字段说明

| 字段 | 含义 |
|-----|------|
| 作品总数 | 账号累计发布的作品数量 |
| 总点赞数 | 所有作品获得的点赞总数 |
| 总收藏数 | 所有作品获得的收藏总数 |
| 近30天发作品数 | 近30天内发布的作品数量 |
| 近30天作品互动量 | 近30天作品的点赞+收藏总和 |
