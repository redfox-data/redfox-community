## 通用反空话校验规则（前置）

生成的内容如果出现以下无意义表述直接打回重写：
- 禁止出现：「要做好定位」「要提升内容质量」「要多互动」「要坚持更新」这类没有具体动作、没有数据支撑的表述
- 所有结论/建议必须同时满足3个要求：①有具体数据支撑（自己的历史数据）②有明确可落地的动作 ③有可验证的效果判断标准

## 各模块内容填充硬约束

### 1. 综合诊断结论模块

| 分档 | 强制输出要素 |
|------|--------------|
| >=80分 | 必须列>=2个已验证可复用的具体动作，每个动作绑定数据 |
| <80分 | 必须先点明薄弱项对应的具体数据问题，再给对应建议 |

**综合诊断结论格式**：
- 一段话概括账号定位、目标用户、核心吸引力、当前阶段
- 可借鉴的优点
- 可成长的地方

### 2. 账号定位模块

- **TA是谁**：基于简介+内容+WebSearch的身份判定，一句话概括
- **心智占位**：TA在用户心智里占了什么位置？
- **价值观锚点**：反复出现的价值判断
- **吸引力类型**：从7种类型中选择最匹配的（可组合）
- **差异化优势**：赛道痛点+你的优势+可强化方向

### 3. 粉丝画像与需求模块

- **粉丝构成**：格式「粉丝数X | 女性XX% | XX-XX岁占比最高 | XX/XX/XX TOP3」
- **核心需求反推**：3行表格（画像特征/推断需求/内容匹配度）

### 4. 选题体系模块

- **选题方向**：基于标题聚类的3-5个方向，结合案例说明
- **表达风格**：理性分析/情绪宣泄/故事讲述/清单体/对话体，结合案例说明
- **叙事手法**：悬念前置/对比反转/痛点共鸣/身份代入，结合案例说明
- **人格一致性**：统一度判断

### 5. 封面风格模块

- **视觉特征**：实拍/插画/文字图/拼图/统一色调，结合案例说明
- **信息层级**：标题位置/字体风格/视觉焦点，结合案例说明
- **一致性**：高/中/低

### 6. 爆文能力模块

- **爆文列表**：标题/发布时间/互动数/超标准倍数 表格
- **近7天平均互动**：评估内容整体传播效果
- **爆文特征分析**：标题规律、内容规律、情绪特点

### 7. 互动规模模块（合并原互动效率）

- **互动率**：近30天作品互动量 / 粉丝数
- **收藏率**：总收藏数 / 总点赞数
- **互动结构**：点赞/收藏占比（接口无分享字段）
- **类型判断**：情绪共鸣型/实用价值型
- **水平衡量**：互动率+收藏率对比中位数参考/优秀值参考

### 8. 更新产能模块

- **频率分析**：日更/周更节奏判断
- **发布时间分析**：集中在什么时段
- **双更日分析**：是否存在双更日及效果
- **水平衡量**：表格格式（周均发布+间隔标准差）

### 9. 行动处方模块

- **问题归因**：维度-指标（当前值）格式
- **具体动作**：3条可量化、有节点、有标准的动作

## 核心字段说明

| 字段 | 数据来源 | 说明 |
|------|---------|------|
| 账号名 | 接口raw_data.nickname | 账号昵称 |
| 账号标识 | 接口raw_data.userAttribute映射 | 映射后的账号标识 |
| 综合评分 | 脚本计算 | 七维度总分(满分100) |
| 各维度满分 | 账号定位10/粉丝画像15/选题体系15/封面风格10/爆文能力15/互动规模20/更新产能15 | 合计100分 |
| 近7天平均互动 | 脚本计算 | 近7天作品平均互动数 |
| 间隔标准差 | 脚本计算 | 发布间隔的稳定性指标 |

## 指标计算逻辑说明

| 指标 | 计算公式 | 数据来源 |
|------|---------|---------|
| 互动率 | 近30天作品互动量 / 粉丝数 * 100% | interactiveCountThirty / fans * 100 |
| 收藏率 | 总收藏数 / 总点赞数 * 100% | collected / liked * 100 |
| 爆文率 | 爆文数 / 近7天发作品数 * 100% | viral_count / len(works) * 100 |
| 爆文标准 | 素人1000/尾部KOL 5000/腰部KOL 10000/头部KOL 20000/明星 20000 | 根据账号标识动态计算 |
| 近7天平均互动 | 近7天作品互动数之和 / 近7天作品数 | 基于作品数据计算 |
| 间隔标准差 | 发布间隔的标准差 | 基于作品时间序列计算 |
