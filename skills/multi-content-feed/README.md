# 全网内容出海信息源 / multi-content-feed

---

## 简介

全网内容出海信息源，每日扫描全平台内容出海爆款作品，按点赞量筛选 Top50，智能聚类题材方向后生成包含平台标签、封面、互动数据与创作洞察的可视化 HTML 日报。支持按平台、关键词、时间范围定向查询。

**核心价值**

- 📊 **全平台一站覆盖**：公众号、抖音、视频号、小红书、快手、B 站六大平台数据一次性获取，无需逐平台查找。
- 🏷️ **智能题材聚类**：基于数据自动归类热门方向，无需预定义关键词，动态反映当日题材分布。
- 📈 **可视化日报**：白色主题 HTML，展示平台彩色标签、封面图、互动数据与作品直达链接，直观易读。
- 🔍 **灵活定向查询**：支持按平台筛选、关键词搜索、自定义时间范围，精准覆盖任意细分方向。

**适用对象**

- 🎬 **内容出海创作者** — 精准把握全平台流量风口，提升选题命中率。
- 📊 **运营人员 / MCN 机构** — 追踪竞品动态与平台热点，优化运营策略。
- 📝 **内容策划 / 数据分析师** — 产出结构化趋势报告，支撑内容战略决策。

---

## 功能特性

### 核心功能

- **全平台爆款发现**：每日扫描六大平台内容出海中按点赞量获取 Top50 作品，精准定位各平台高热度内容。
- **智能题材聚类**：基于 API 返回的主题标签自动归类，题材分类完全由当日数据决定，无需预置关键词。
- **可视化 HTML 日报**：白色主题页面，每条作品展示对应平台彩色标签、封面图、点赞/评论/分享数据与作品直达链接。
- **多维度定向查询**：默认查询全平台，可按关键词和平台组合筛选，灵活覆盖任意细分方向。
- **创作趋势洞察**：分析各平台内容风格、跨平台题材对比、平台差异化策略，深度挖掘创作规律。
- **一键每日订阅**：开启后日报自动保存至本地文件夹，无需手动操作。

### 特色亮点

- ⚡ **智能日期判断**：内置数据更新时间常量，自动检测目标日期是否在可用区间，避免无效查询消耗积分。
- 🌐 **全平台排序适配**：公众号按阅读数排序，其余平台按点赞数排序；作品链接直接来自数据源，无需手动拼接。
- 🏷️ **平台彩色标签**：HTML 日报按平台标识色区分——公众号绿、抖音黑、视频号橙、小红书红、快手橙、B 站蓝，一目了然。
- 🔒 **安全可靠**：API Key 仅通过环境变量获取，禁止硬编码，数据来源唯一可追溯。

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 获取最新日报 | 「生成今天的内容出海日报」 | 自动计算最新可用日期，生成全平台 Top50 可视化日报 |
| 查询历史日期 | 「查一下 6 月 10 号的内容出海日报」 | 直接获取指定日期的历史爆款数据 |
| 按关键词搜索 | 「搜一下品牌出海相关的爆款」 | 模糊匹配标题与名称，输出该方向的 Top50 作品 |
| 指定平台查询 | 「只看小红书和抖音的内容出海爆款」 | 过滤指定平台，聚焦核心渠道 |
| 开启每日订阅 | 「帮我开启每日内容出海日报」 | 日报自动保存至本地，无需每次手动操作 |

### 输出示例

运行后将获得：

- **HTML 可视化日报**：白色主题页面，按平台分区展示 Top8 热门作品，含封面图、互动数据、彩色平台标签与作品直达链接，自动打开浏览器预览。
- **终端摘要分析**：包含数据全景（六平台指标一览）、跨平台差异洞察（题材矩阵与核心洞察）、趋势判断及验证（含可执行行动建议）三大分析版块。

## 重要数据说明

- 数据每日 **15:00** 更新前一天的数据。
- 15:00 之前最新可用日期为前天（T-2），15:00 之后为昨天（T-1）。
- 目标日期无数据时需先告知用户并等待确认，不可自动调用数据接口。
