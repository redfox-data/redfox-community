# 全网内容出海信息源 / multi-content-feed

---

## 简介

全网内容出海信息源是一款全平台内容出海爆款追踪工具，每日自动扫描公众号、抖音、视频号、小红书、快手、B站六大平台的内容出海爆款作品，按互动量筛选 Top50，智能聚类题材方向，生成包含平台标签、封面图、互动数据与创作洞察的可视化日报。

**核心价值**

- 📊 **一站式全平台覆盖**：同时监控公众号、抖音、视频号、小红书、快手、B站六大平台，无需逐一打开各平台查看
- 🏷️ **智能题材聚类**：基于数据自动归类题材方向，动态反映当日热点分布，无需手动预定义关键词
- 📈 **深度创作洞察**：不仅展示爆款作品，还提供跨平台差异分析、趋势判断与行动建议
- 🎨 **可视化日报**：白色主题 HTML 日报，平台彩色标签、封面图、互动数据一目了然

**适用对象**

- 🎬 内容出海创作者 — 精准把握全平台流量风口，提升选题命中率
- 📊 内容出海运营/MCN机构 — 追踪竞品表现，及时掌握行业动态
- 📋 内容策划/数据分析师 — 产出结构化趋势报告，支撑内容战略决策

---

## 功能特性

### 核心功能

- 全平台爆款发现：从公众号、抖音、视频号、小红书、快手、B站按互动量获取 Top50 作品
- 智能题材聚类：基于作品话题字段自动归类，题材分类完全由数据决定
- 定向查询：支持按关键词、平台、时间范围灵活检索
- 可视化日报：白色主题 HTML，含平台彩色标签、封面图、互动数据与作品直链
- 创作洞察：分析各平台内容风格、跨平台题材对比、平台差异化策略
- 一键订阅：开启后每日自动产出日报，本地文件夹持续累积

### 特色亮点

- ⚡ 智能日期判断：内置数据更新时间逻辑，调用前自动检测目标日期是否可用
- 🌐 全平台适配：六大平台各自按阅读数或互动数排序，作品链接直接可用
- 🏷️ 平台标签：日报中每条作品展示彩色平台标识，一眼区分来源
- 🔒 安全可靠：密钥通过环境变量获取，禁止硬编码

## 密钥获取与安全说明

- 本技能需要使用环境变量：`REDFOX_API_KEY`。
- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?souce=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?souce=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 获取最新日报 | "生成今天的内容出海日报" / "查询最新日报" | 自动获取最新有数据的日期，生成全平台日报 |
| 按日期查询 | "查询 2026-06-10 的内容出海日报" | 查询指定历史日期的爆款数据 |
| 按关键词搜索 | "搜索品牌出海相关的爆款" | 在六大平台中模糊匹配关键词 |
| 只看某平台 | "只看小红书和抖音的内容出海" | 仅查询指定平台的数据 |
| 开启每日推送 | "开启内容出海日报订阅" | 每日自动产出日报到本地文件夹 |

### 输出示例

查询后生成两部分内容：

1. **HTML 可视化日报**：包含平台概览表、各平台 Top8 作品卡片（封面图 + 标题 + 作者 + 互动数据）、题材标签云，自动在浏览器中打开
2. **终端摘要报告**：数据全景表（各平台作品数、互动分布、头部天花板）、跨平台差异洞察、趋势判断与行动建议

## 重要数据说明

- **数据更新时间**：每日 15:00 更新前一天的数据
- **数据覆盖范围**：公众号、抖音、视频号、小红书、快手、B站六大平台，每个平台取互动量 Top50
- **排序规则**：公众号按阅读数排序，其余平台按互动量排序
- **日期约束**：15:00 前最新可用日期为前天，15:00 后为昨天；查询目标日期无数据时会先提示确认
