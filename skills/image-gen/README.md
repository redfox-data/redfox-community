# GPT-image2 / image-gen

---

## 简介

调用 OpenAI 最新的 **gpt-image-2** 模型生成高质量图片，支持文生图与图生图双模式。内置免费额度，粘贴提示词就能用。

**核心价值**

- **开箱即用**：内置公共 API Key，约 **10000 次免费额度**，无需额外订阅或绑定信用卡。
- **双模式创作**：文生图从零创作，图生图基于参考图编辑，`--image` 一个参数切换。
- **批量高效**：单次最多生成 10 张图片，支持透明背景和多种格式输出。

**适用对象**

- 🎨 **设计师** — 快速生成概念图、Logo、插画素材。
- 📱 **内容创作者** — 批量产出社交媒体配图、封面图。
- 🛒 **电商运营** — 零成本生成产品展示图和场景图。
- 💡 **AI 爱好者** — 体验 gpt-image-2 模型的强大图像生成能力。

---

## 功能特性

### 核心功能

- **文生图**：输入提示词，从零生成全新图片。
- **图生图**：上传参考图 + 提示词，基于原图进行编辑和风格转换。
- **批量生成**：单次最多生成 10 张图片，适合批量素材需求。
- **多格式输出**：支持 PNG、JPEG、WebP 三种格式。
- **透明背景**：支持 PNG / WebP 透明背景输出，适合 Logo 和图标设计。
- **保真度控制**：图生图支持 high / low 保真度，微调或大幅改动自由选择。
- **多分辨率支持**：快速档（1024×1792 等）和高清档（2048×2048 等），按需选择。
- **任务管理**：支持仅提交任务获取 taskId，稍后查询结果。

## 使用指南

直接用自然语言描述你想要的图片，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 文字生成图片 | 「帮我画一只橘色猫咪坐在窗台上看夕阳」 | 自动提交任务并下载生成的图片 |
| 生成横版高清图 | 「未来城市天际线，横版高清」 | 以 1792×1024 横版分辨率生成图片 |
| 透明背景 Logo | 「极简猫咪 Logo，扁平设计，透明背景」 | 生成透明背景的 PNG 图片 |
| 批量生成 | 「一组扁平风格图标，要 4 张」 | 一次生成 4 张统一风格的图片 |
| 参考图编辑 | 「把猫咪改成白色，背景换成星空」 | 上传参考图后基于原图进行编辑 |
| 风格迁移 | 「把这张照片改成赛博朋克风格，高保真」 | 基于参考图进行高保真度的风格转换 |
