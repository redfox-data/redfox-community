# GPT - image2 / image-gen

---

## 简介

调用 OpenAI gpt-image-2 模型生成高质量图片，支持文生图与图生图，内置公共 API Key，开箱即用。

**核心价值**

- **双模式生成**：文生图 + 图生图，一个工具覆盖创作与编辑场景。
- **批量交付**：单次最多生成 10 张，参数化控制尺寸/格式/背景。
- **零配置启动**：内置公共 API Key（约 10000 次免费额度），直接用。

**适用对象**

- 🎨 **设计师 / 创作者** — 快速生成创意素材、logo、插画。
- 📱 **内容运营** — 批量产出配图、产品图、社交媒体素材。
- 💼 **产品经理** — 快速出原型图、概念可视化。

---

## 功能特性

### 核心功能

- **文生图**：输入提示词，生成全新图片。
- **图生图**：上传参考图 + 提示词，基于原图编辑生成（支持高/低保真度控制）。
- **批量生成**：单次最多 10 张，支持自定义尺寸（1024x1024 / 1024x1792 / 1792x1024）。
- **多格式输出**：PNG、JPEG、WebP，支持透明背景。
- **进度显示**：自动轮询等待，实时显示生成进度。

### 特色亮点

- **内置免费额度**：约 10000 次，无需订阅或绑定信用卡。
- **透明背景**：搭配 PNG/WebP 格式，一键生成透明底图。

## API Key 说明

内置公共 API Key，约 **10000 次免费额度**，零配置直接使用。额度用完后可前往 [RedFoxHub](https://redfox.hk/login?source=github) 注册获取个人 API Token：

| 方式                 | 命令                                                                                     |
| -------------------- | ---------------------------------------------------------------------------------------- |
| **环境变量**（推荐） | `export REDFOX_API_KEY=ak_你的密钥`                                                      |
| **命令行参数**       | `python3 "$SKILL_PATH/assets/imagegen.py" "prompt" --api-key ak_你的密钥`                |
| **配置文件**         | `mkdir -p ~/.qoder/apis && echo '{"api_key":"ak_你的密钥"}' > ~/.qoder/apis/redfox.json` |

## 使用指南

用自然语言描述想要的图片即可，无需记忆命令参数。

### 常用说法速查

| 意图     | 示例话术                               | 效果               |
| -------- | -------------------------------------- | ------------------ |
| 文生图   | 「帮我画一只橘色的猫咪坐在窗台看夕阳」 | 生成一张全新图片   |
| 横版图片 | 「生成一张赛博朋克城市天际线的横版图」 | 1792x1024 横版输出 |
| 透明背景 | 「生成一个极简猫咪 logo，透明背景」    | PNG 格式透明底图   |
| 图生图   | 「把这张图的猫改成白色，背景换星空」   | 基于参考图编辑生成 |
| 批量生成 | 「生成 4 个扁平风格图标」              | 一次输出 4 张图片  |
