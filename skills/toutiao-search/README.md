# 今日头条爆款内容查询 / toutiao-search

---

## 简介

输入关键词就能搜到今日头条刚发布的作品，能按阅读量或发布时间排序、限定最近几小时内内容，给到你的都是平台刚更新的新鲜内容，不会拿旧缓存数据糊弄你。

**核心价值**

- **内容新鲜不滞后**：每次搜索都直接拉取平台最新发布的作品，只给刚发的内容，不用老旧存档、缓存数据。
- **筛选方式灵活好用**：能按阅读量或发布时间排序，还能限定最近几小时内内容，自由搭配条件查找。
- **图文视频都能搜**：图文和视频作品一网打尽，想看哪种都能找到，不会遗漏任何类型。
- **翻页查看不遗漏**：结果多的时候可以一页页翻着看，不遗漏任何一条内容。
- **数据指标一目了然**：阅读量、点赞、评论、转发等互动数据直观展示，还能自动导出 CSV 和交互式 HTML 报告，下载即用。

**适用对象**

- 🔍 **内容创作者** — 快速了解今日头条上同赛道最新趋势与热门选题，获取创作灵感。
- 📊 **运营 / 数据分析** — 追踪关键词相关内容在头条的数据表现，掌握流量趋势。
- 🏢 **品牌 / MCN** — 监控竞品与行业关键词在今日头条的最新内容表现，辅助策略决策。

---

## 功能特性

### 核心功能

- **关键词搜索**：输入任意关键词搜索今日头条最新爆款作品。
- **双排序切换**：按阅读量（`views`）或发布时间（`time`）降序排列，快速定位高质量或最新内容。
- **时间过滤**：`--hours N` 限定最近 N 小时内的新鲜内容，拒绝过期数据。
- **视频筛选**：`--video-only` 只返回视频类作品，精准找视频爆款。
- **翻页浏览**：`--pages N` 控制翻页次数，每页约 10 条，默认 3 页。
- **终端表格输出**：标题、作者、阅读、点赞、评论、转发、发布时间、类型一览无余。
- **CSV 自动导出**：UTF-8 BOM 编码 CSV，Excel 直接打开不乱码。
- **HTML 交互报告**：深色主题，图文/视频标签区分，作品卡片点击跳转原文，支持页内搜索。

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 搜某个话题内容 | 「搜一下今日头条 AI 相关内容」 | 按阅读量排序搜索并展示结果 |
| 指定排序方式 | 「按最新发布搜索大模型相关作品」 | 切换为按发布时间排序后搜索 |
| 限定时间范围 | 「搜索最近 24 小时新能源相关作品」 | 设置 `--hours 24`，只返回一天内作品 |
| 只看视频内容 | 「搜一下测评类的视频」 | 开启 `--video-only`，只返回视频作品 |
| 翻页查看更多 | 「多翻几页」/「看更多结果」 | 增加 `--pages` 翻页获取更多数据 |
| 导出数据 | 「导出 CSV」/「生成 HTML 报告」 | 自动生成 CSV 和交互式 HTML 报告 |
