# 小红书对标账号输出模版

## 输出顺序
1. 提示语（只显示有数据的组）
2. 同阶对标表格（有数据才展示）
3. 高阶标杆表格（有数据才展示）
4. 分析总结（有数据才展示）
5. 订阅服务
6. 展示HTML文件（AI读取并直接展示HTML文件）

## 推荐理由生成规则

推荐理由需口语化描述，综合分析以下维度：
1. **内容方向**：专注测评方向、干货内容为主、平价好物推荐、避雷种草风格、剧情类内容、日常vlog风格等
2. **更新频率**：日更博主更新很勤快、更新稳定内容持续输出、有在持续更新
3. **互动表现**：互动量很高内容质量不错、互动表现不错、有一定互动基础
4. **爆文案例**：展示爆文标题和互动数

示例：
- "日常vlog风格，更新稳定，内容持续输出，互动量很高，内容质量不错"
- "剧情类内容，有在持续更新，爆文「整蛊隔壁桌|遇到急性子女...」获3665互动"
- "干货内容为主，有在持续更新，互动表现不错"

## 数值格式化规则
- 数值 < 10000：直接展示原值
- 数值 >= 10000：格式化为 "X.Xw"（如 15000 -> 1.5w）
