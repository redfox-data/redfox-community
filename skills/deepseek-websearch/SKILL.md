# Deepseek WebSearch

## 📝 简介

Deepseek WebSearch 是一款 AI 联网搜索助手，通过红狐 API 调用 Deepseek 联网搜索能力，提交查询后自动轮询等待结果返回。

适用于需要联网实时搜索、AI 辅助信息检索的场景，帮助用户快速获取最新、准确的信息。

---

## ✨ 功能特性

| 功能模块 | 能力描述 | 核心价值 |
|---------|---------|---------|
| 异步提交 | 提交搜索关键词到 Deepseek WebSearch，返回 taskId | 无需等待，立即获取任务标识 |
| 自动轮询 | 每 5 秒查询任务状态，最多等待 5 分钟 | 自动等待结果，无需手动查询 |
| 进度反馈 | 轮询期间通过 stderr 实时输出等待状态 | 用户可知搜索进度 |
| 结果输出 | 状态变为 succeeded 后输出完整 JSON | 结构化数据便于 Agent 解析展示 |
| 异常处理 | 覆盖 API Key 缺失、提交失败、超时、任务失败四种场景 | 每种异常有明确提示和处理 |

## 🔑 一键安装

### 前置条件

- 已注册红狐 hub 账号并获取 API Key
- 本地已安装 Python 3 环境

### 安装步骤

1. **获取 API Key**

   前往 [红狐hub](https://redfox.hk/settings/api-keys?source=github) 注册并获取 `REDFOX_API_KEY`。

2. **配置环境变量**

   ```bash
   # 方式一：配置文件（如 OpenClaw 的 ~/.openclaw/openclaw.json）
   { "env": { "REDFOX_API_KEY": "ak_xxxx..." } }

   # 方式二：终端环境变量
   export REDFOX_API_KEY="ak_xxxx..."
   ```

3. **安装依赖**

   ```bash
   pip3 install requests
   ```

## 📖 使用指南

直接用自然语言描述你的搜索需求即可，无需记忆固定命令。

### 基础使用

#### 1. 发起搜索

告诉助手你想搜索的内容：

> 用户：帮我搜索一下最新的 AI 大模型发展趋势
>
> 助手：正在为您调用 Deepseek WebSearch 搜索...

#### 2. 等待结果

脚本会自动提交搜索并轮询等待结果，期间会实时反馈进度：

```
⏳ 已提交搜索请求 (taskId: xxx)，正在等待 Deepseek WebSearch 返回结果…
   ⏳ 第 1 次轮询，状态: queued，继续等待…
   ⏳ 第 2 次轮询，状态: running，继续等待…
```

#### 3. 查看结果

搜索完成后，助手会整理 Deepseek 的回答文本和引用来源，结构化展示给你。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 实时信息查询 | 「搜索一下今天的科技新闻」 | 调用 Deepseek 联网搜索，获取最新资讯 |
| 专业知识检索 | 「帮我查一下量子计算的最新进展」 | 搜索并整理专业领域的最新研究成果 |
| 事实验证 | 「搜索验证一下这个说法是否准确」 | 通过联网搜索核实信息准确性 |

## ⚠️ 错误处理

| 情况 | 处理方式 |
|------|---------|
| 未配置 `REDFOX_API_KEY` | 提示用户前往 [红狐hub](https://redfox.hk/settings/api-keys?source=github) 获取 API Key |
| 服务返回 502 | 提示可能存在网络不稳定问题，请稍后重试 |
| 提交失败 | 输出 HTTP 状态码及错误信息，建议用户重试 |
| 轮询超时（5 分钟） | 提示超时，建议稍后重试 |
| 任务失败（status=failed） | 输出 failReason 详情 |
