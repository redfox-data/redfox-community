# AI 公众号信息源 / gzh-ai-feed

---

## 简介

每日自动扫描全网 AI 公众号，按阅读量筛选最火爆款内容，智能聚类后生成精美 HTML 日报，内置公共 API Key，开箱即用。

**核心价值**

- **智能聚类**：自动发现当天热门话题方向，无需手动分类。
- **可视化日报**：深色主题 HTML，封面图、互动数据、文章直链一应俱全。
- **零配置启动**：内置公共 API Key（约 10000 次免费额度），直接用。

**适用对象**

- 📰 **AI 从业者** — 每日获取 AI 领域热点，不错过重要动态。
- 📊 **内容运营** — 追踪 AI 赛道公众号爆款，辅助选题决策。
- 🧠 **研究者 / 学生** — 聚合 AI 前沿信息，降低信息筛选成本。

---

## 功能特性

### 核心功能

- **爆款发现**：从 200+ 篇 AI 公众号文章中按阅读量筛选最火内容。
- **智能聚类**：自动识别话题方向（Agent、大模型、AI 绘画…），每天分类由内容决定。
- **终端表格**：分类 + 标题 + 作者 + 阅读/点赞/评论数，一目了然。
- **可视化日报**：深色主题 HTML，封面图、互动数据、文章直链、日期导航。
- **全库搜索**：日报页面内置搜索框，实时检索全量文章库。
- **一键订阅**：`--subscribe` 开启每日自动产出。

## API Key 说明

内置公共 API Key，约 **10000 次免费额度**，零配置直接使用。额度用完后可前往 [RedFoxHub](https://redfox.hk/login?source=github) 注册获取个人 API Token：

| 方式 | 命令 |
|------|------|
| **环境变量**（推荐） | `export REDFOX_API_KEY=ak_你的密钥` |
| **命令行参数** | `--api-key ak_你的密钥` |
| **配置文件** | `echo '{"api_key":"ak_你的密钥"}' > ~/.qoder/apis/redfox.json` |

## 使用指南

直接描述你想看的 AI 日报即可，无需记忆命令。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|
| 今日日报 | 「给我看看今天的 AI 日报」 | 生成今日爆款 HTML 日报 |
| 指定方向 | 「我关注 AI Agent 和 RAG 方向」 | 按自定义关键词生成日报 |
| 查看历史 | 「给我 5 月 26 号的 AI 日报」 | 生成指定日期的日报 |
| 开启订阅 | 「每天帮我自动生成 AI 日报」 | 安装每日定时任务 |
