# 公众号搜索爬虫 / gzh-search

---

## 简介

输入关键词搜索微信公众号文章，终端表格展示 + CSV 导出 + 交互式 HTML 报告，内置公共 API Key，开箱即用。

**核心价值**

- **智能评分排序**：三因子（相关性 + 热度 + 时效）综合评分，精准定位优质内容。
- **双格式输出**：终端实时查看 + CSV/HTML 持久化存档，满足不同场景。
- **零配置启动**：内置公共 API Key（约 10000 次免费额度），直接用。

**适用对象**

- 📊 **内容运营** — 追踪行业热点，发现爆款选题。
- 🏢 **品牌 / 市场团队** — 竞品内容分析，话题舆情监控。
- 📝 **自媒体创作者** — 搜集素材灵感，了解赛道动态。

---

## 功能特性

### 核心功能

- **关键词搜索**：实时查询全量微信公众号文章。
- **智能评分排序**：三因子（相关性 + 热度 + 时效）综合评分，同分按阅读量降序。
- **终端表格**：标题、作者、阅读、点赞、分享、收藏、发布时间、文章链接。
- **CSV 导出**：自动生成 UTF-8 BOM 编码的 CSV。
- **HTML 交互报告**：内置搜索框，封面图展示，点击卡片跳转原文。
- **分层响应**：充足正常浏览 / 较少引导宽泛搜索 / 无结果热门兜底。

## API Key 说明

内置公共 API Key，约 **10000 次免费额度**，零配置直接使用。额度用完后可前往 [RedFoxHub](https://redfox.hk/login?source=github) 注册获取个人 API Token：

| 方式 | 命令 |
|------|------|
| **环境变量**（推荐） | `export REDFOX_API_KEY=ak_你的密钥` |
| **命令行参数** | `--api-key ak_你的密钥` |
| **配置文件** | `echo '{"api_key":"ak_你的密钥"}' > ~/.qoder/apis/redfox.json` |

直接描述你想搜的公众号内容即可。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|----------|------|

### 重要数据说明

- 搜索范围覆盖**腰部以上公众号**近 30 天内发布的文章。
- 关键词限制：**不超过 10 个字符**，超出会提示精简。
- 搜特定公众号全部历史文章请使用 `gzh-subscribe` 技能。
