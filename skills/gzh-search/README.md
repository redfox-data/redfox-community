# 公众号搜索爬虫 / gzh-search-crawler

---

## 简介

输入关键词搜索微信公众号文章，终端表格展示并自动导出 CSV + 交互式 HTML 报告。

**核心价值**

- **关键词搜索**：实时查询全量微信公众号文章，覆盖腰部以上公众号近 30 天内容
- **智能评分排序**：综合关键词相关性、阅读热度、时效新鲜度三维评分
- **多格式输出**：终端表格 + CSV 导出 + 交互式 HTML 报告，满足不同场景
- **交互式报告**：HTML 页面内置搜索框，支持封面图展示、卡片点击跳转原文

**适用对象**

- 🔍 **行业研究员** — 追踪行业热点，批量导出数据做趋势分析
- 📝 **内容创作者** — 发现爆款选题，搜集素材灵感
- 🏢 **竞品分析师** — 监控竞争对手公众号内容策略

---

## 功能特性

### 核心功能

- **关键词搜索**：输入任意关键词（不超过 10 字），实时搜索公众号文章
- **智能排序**：三维评分（相关性 + 热度 + 时效），同分按阅读量降序
- **分层降级**：充足时正常浏览 / 较少时引导宽泛搜索 / 无结果时热门兜底
- **终端表格**：标题、作者、阅读、点赞、分享、收藏、发布时间、文章链接
- **CSV 导出**：自动生成 UTF-8 BOM 编码 CSV，便于数据分析
- **HTML 报告**：内置搜索框（300ms 防抖），封面图展示，分页加载

- `REDFOX_API_KEY` 由 [红狐 hub](https://redfox.hk/settings/api-keys?source=github) (`https://redfox.hk`)提供。
- 请前往 [红狐 hub](https://redfox.hk?source=github) 注册账号，获取 `REDFOX_API_KEY`。
- 配置设备环境变量 `REDFOX_API_KEY` 后使用本技能。
- 在提供密钥前，请先确认密钥来源、可用范围、有效期及是否支持重置/撤销。
- 禁止在代码、提示词、日志或输出文件中硬编码/明文暴露密钥。

直接用自然语言说出你想搜的关键词即可。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
|------|---------|------|
| 搜索热点 | 「搜一下 AI 相关的公众号文章」 | 返回 AI 相关热门文章，按评分排序 |

| 素材搜集 | 「搜小红书运营的爆款文章」 | 获取同赛道选题和写作角度 |
| 趋势研究 | 「搜 2026 经济相关的公众号文章」 | 批量导出数据做研究报告 |
